**修复方案**:

```rust
// 修改后

```

### 2. Select 不是迭代器 (约 15 处)

**错误模式**:
```rust
let total = query.clone().count(&*self.db).await?;
```

**问题**: SeaORM 的 `Select` 没有实现 `Iterator` trait

**修复方案**:
```rust
// 方案 1: 使用 count_by_id (如果适用)
let total = query.clone().count(&*self.db).await?;

// 方案 2: 使用 count 方法 (需要确认正确 API)
let total = query.clone().count(&*self.db).await?;

// 方案 3: 获取所有数据后计算长度
let items = query.clone().all(&*self.db).await?;
let total = items.len() as u64;
```
