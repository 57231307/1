- ⚠️ `sales_orders.customer_id` → 无关联表 (信息孤岛)

#### 发现严重问题 🔴

❌ **问题 1**: 创建订单时**不检查库存**  
**位置**: `backend/src/services/sales_service.rs:236-345`  
**影响**: 可以创建库存不足的订单，导致无法发货  
**建议**: 
```rust
// 在 create_order() 中添加库存检查
for item in &request.items {
    let stock = InventoryStockEntity::find()
        .filter(inventory_stock::Column::ProductId.eq(item.product_id))
        .one(&txn)
        .await?;
    
    if stock.quantity < item.quantity {
        txn.rollback().await?;
        return Err(DbErr::Custom(format!(
            "产品 {} 库存不足，当前库存：{}", 
            item.product_id, stock.quantity
        )));
    }
}
```

❌ **问题 2**: 审核订单时**不锁定库存**  
**位置**: `backend/src/services/sales_service.rs` (缺少 approve_order 实现)  
**影响**: 可能导致超卖（多个订单同时审核，库存被重复扣减）  
**建议**: 审核时锁定库存，创建库存预留记录

❌ **问题 3**: 发货时**不扣减库存**  
**位置**: `backend/src/services/sales_service.rs:439-469`  
**影响**: 订单状态变为已发货，但库存数量未变，导致账实不符  
**建议**: 
```rust
// 在 ship_order() 中扣减库存
for item in &order.items {
    let stock = InventoryStockEntity::find_by_id(item.stock_id)
        .one(&txn)
        .await?;
    
    let updated_stock = inventory_stock::ActiveModel {
        id: ActiveValue::Unchanged(stock.id),
        quantity: ActiveValue::Set(stock.quantity - item.quantity),
        updated_at: ActiveValue::Set(Utc::now()),
        ..Default::default()
    };
    updated_stock.update(&txn).await?;
}
```

❌ **问题 4**: 缺少客户管理模块  
**影响**: `sales_orders.customer_id` 是孤立字段，没有客户表关联  
**建议**: 创建 `customers` 表，实现客户管理功能

## 六、改进建议实施计划

### 6.1 第一阶段（1-2 天）- 修复严重问题

#### 任务 1: 销售订单库存检查
```rust
// 在 create_order() 中添加
async fn check_inventory(&self, items: &[OrderItemRequest]) -> Result<(), DbErr> {
    for item in items {
        let stock = InventoryStockEntity::find()
            .filter(inventory_stock::Column::ProductId.eq(item.product_id))
            .one(&self.db)
            .await?;
        
        match stock {
            Some(s) if s.quantity >= item.quantity => continue,
            Some(s) => return Err(DbErr::Custom(
                format!("产品 {} 库存不足", item.product_id)
            )),
            None => return Err(DbErr::Custom(
                format!("产品 {} 没有库存记录", item.product_id)
            )),
        }
    }
    Ok(())
}
```

#### 任务 2: 销售订单库存锁定
```rust

async fn lock_inventory(&self, order_id: i32) -> Result<(), DbErr> {
```
