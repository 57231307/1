#### 任务1: 修复采购订单API响应格式
**状态**: ⏳ 待执行  
**预计时间**: 2小时  
**负责人**: 后端开发

**问题分析**:
- 错误: "invalid type: map, expected a sequence"
- 原因: 后端返回map结构,前端期望array
- 影响: 采购模块完全无法使用

**修复步骤**:
1. 检查 `backend/src/handlers/purchase_order_handler.rs`
2. 找到列表查询handler
3. 确认当前返回格式
4. 修改为正确的格式
5. 重新编译并测试

**验证方法**:
```bash
curl http://localhost:8082/api/v1/erp/purchases/orders | python3 -m json.tool
```
