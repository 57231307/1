
### 1.2 扫描模式

| 模式 | 描述 | N+1 风险 |
|------|------|----------|
| `find_with_related` | SeaORM 关系加载（通常安全） | 中（需确认关系数） |
| `find_related` | 反向关系加载（可能 N+1） | 高（需 preload 验证） |
| `for ... in ...` + `service::*` | 循环中调用其他 Service | 高（典型 N+1） |
| `for ... in ...` + `find_*` | 循环中查询数据库 | 极高（明确 N+1） |

### 1.3 扫描命令

```bash
cd /workspace/backend
bash scripts/p2-2-n1-scan.sh > /tmp/p2-2-n1-list.md
```
