## 十、实施方法论

### 10.1 subagent-driven 模式

冰溪 ERP 项目采用 **subagent-driven（子代理驱动）** 模式，由总代理（项目经理）统一调度，子代理（专项执行）独立完成具体任务。

```
┌──────────────────────────────────────────────────────────┐
│              总代理（冰溪 ERP 项目经理）                    │
│    - 项目分解 / 阶段规划 / 任务派发 / 质量验收              │
│    - 7 波次调度 / 35 PR 编排 / 全局把控                     │
└────────────────────────┬─────────────────────────────────┘
                         │ 派工
        ┌────────────────┼────────────────┐
        ▼                ▼                ▼
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  P0 子代理   │ │  P1 子代理   │ │  P2-P7 子代理│
│  行业子模块  │ │  代码清理    │ │  性能/AI/运维│
│  5 PR        │ │  8 PR        │ │  22 PR       │
└──────────────┘ └──────────────┘ └──────────────┘
```
