## 一、面料行业基础信息

### 1.1 行业定位

面料行业是纺织产业链的**中游环节**，连接上游纤维原料生产（棉/毛/丝/麻/化纤）与下游终端制造（服装/家纺/产业用布）。

```
纤维原料 → 纺纱 → 织布（坯布）→ 染整（染色/印花/后整理）→ 成品面料 → 服装/家纺
```

面料 ERP 覆盖范围：**原材料采购 → 坯布织造 → 染色印花 → 后整理 → 仓储物流 → 销售回款** 全流程。

### 1.2 面料行业核心特性（区别于通用 ERP）

| 特性 | 通用 ERP | 面料行业 ERP |
|------|----------|--------------|
| SKU 维度 | 物料编码单一 | 面料编码 + 色号 + 缸号 + 批号（匹号）四维标识 |
| 计量单位 | 单一单位 | 米/公斤/码/匹四单位动态换算 |
| 批次管理 | 批次属性单一 | 缸号/色号/幅宽/克重/色牢度/手感等多维特征 |
| 计价模式 | 标准计价 | 按米价/公斤价/码价 + 加工费 + 损耗率 + 阶梯折扣 |
| 质检标准 | 通用合格/不合格 | 四分制/十分制疵点评分 + 色差 ΔE + A/B/C 级判定 |
| 库存管理 | 数量管理 | 数量 + 颜色 + 缸号 + 库位 + 质检状态多维管理 |
| 成本核算 | 平均成本法 | 按缸号实际成本法（每缸独立核算） |

### 1.3 面料核心属性（27 项检测参数）

**物理维度**：
- 克重（g/m²）：单位面积重量，影响成本和定价
- 幅宽/门幅（cm）：面料宽度，影响用料计算
- 缩率（%）：水洗缩水率，弹力面料关键参数
- 色牢度等级：耐水洗/耐摩擦/耐光色牢度
- 经纬密度：经纱×纬纱根数
- 成分百分比（精确至 0.1%）：棉/涤/粘胶/氨纶等
- 是否含氨纶及弹性回复率
- 纱支：纱线粗细（如 40S）

**工艺维度**：
- 染整工艺路线编号（如"活性染色-高温固色-柔软定型"）
- 适用裁床类型（自动铺布机/人工铺布）
- 最小起订量（MOQ）与最小发运量（MOV）

**商务维度**：
- 供应商批次追溯深度（原厂批号 + 入库批号 + 染厂缸号 三级关联）

**财务维度**：
- 存货科目、税目编码
- 折旧计提方式（坯布按实际成本法、辅料按计划成本法）

## 二、面料行业核心概念体系

### 2.1 四层级联关系（v14 复审核心）

```
面料（Fabric/Product）
  └── 颜色（Color）              ← 一个面料有多个颜色（核心特性）
       ├── 色号（ColorNo）        ← 颜色唯一编码，如 C001
       ├── 色卡（ColorCard）      ← 颜色实物样卡
       └── 缸号（DyeLotNo）       ← 每个颜色每批次染色有独立缸号
            ├── 批号/匹号（BatchNo）← 同一缸号下可分多个批次（匹号）
            ├── 库存数量          ← 按缸号/批号独立核算
            ├── 质检结果          ← 按缸号/批号独立检验
            ├── 成本单价          ← 按缸号/批号独立核算（实际成本）
            └── 库位              ← 按缸号分区存放
```

### 2.2 色号/缸号/卷号/匹号定义

| 术语 | 英文 | 定义 | 业务作用 |
|------|------|------|----------|
| 色号 | ColorNo | 面料基础颜色编码 | 区分基础配色 |
| 缸号 | DyeLotNo/VatNo | 面料染色批次 | **同色不同缸存在肉眼可见色差，裁床严禁不同缸号面料混铺**，是色差管控核心 |
| 卷号 | RollNo | 单匹独立面料编号 | 面料库存最小管理单元 |
| 匹号 | BatchNo | 同一缸号下分多个匹号 | 每匹有独立长度和质检结果 |

### 2.3 关键业务约束（v14 复审强制执行）

**约束 1：匹号唯一约束**
- **同一缸号（dye_lot_no）下不能有相同的匹号（batch_no）**
- DB 层：`UNIQUE(dye_lot_no, batch_no)` 唯一索引
- 业务语义：一个缸号代表一次染色，同一缸内分多匹出布，每匹有独立匹号；匹号重复会导致库存和成本核算混乱

**约束 2：面料-颜色关联约束**
- `UNIQUE(product_id, color_id)` —— 同一面料+颜色组合唯一

**约束 3：缸号-颜色关联约束**
- `UNIQUE(color_id, dye_lot_no)` —— 缸号在颜色范围内唯一

**约束 4：库存四维标识**
- 库存唯一标识 = `product_id + color_id + dye_lot_no + batch_no`（联合唯一索引）

**约束 5：缸号同订单校验**
- 出库时，同一订单必须使用相同缸号的面料
- 系统校验订单中所有该面料是否来自同一批次，不一致则报警提示

### 2.4 SKU 公式

```
面料 SKU = 面料编码 + 色号 + 缸号 + 匹号
服装 SKU = 款式 + 颜色 + 尺码
```

## 三、染整工艺完整流程

### 3.1 染整工艺 10 道工序

```
配布 → 精练（前处理）→ 漂白（前处理）→ 染色 → 对色
  → 理布 → 烘干 → 定型 → 成品对色 → 成检
```

**详细工序说明**：

1. **配布**：按生产订单领用坯布，分配到染缸
2. **精练**：去除坯布上的杂质、浆料、油剂
3. **漂白**：去除天然色素，提高染色鲜艳度
4. **染色**：按染色配方投染染料和助剂，记录温度曲线、时间、pH 值
5. **对色**：实验室对色，比对标准色卡，判定色差 ΔE
6. **理布**：整理布面，去除折痕
7. **烘干**：去除水分，控制缩率
8. **定型**：高温定型，稳定尺寸
9. **成品对色**：成品布对色，确认色差合格
10. **成检**：成品检验，按疵点评分（四分制/十分制）

### 3.2 缸号全生命周期追踪

```
投缸 → 染色 → 皂洗 → 固色 → 脱水 → 烘干 → 验布 → 入库
```

每一环节的操作均需通过 PDA 扫码或工控终端确认，系统自动捕获：
- 时间戳
- 操作人
- 设备 ID
- 实时采集参数（红外测温仪读数、在线色差仪 ΔE 值）

**缸号承载信息**（远超普通生产编号）：
- 坯布来源（供应商 + 批次）
- 染色配方（染料 + 助剂 + 工艺参数）
- 操作班组
- 设备机台
- 水电气实耗
- 首件确认结果
- 全流程质检记录
- 最终落布重量
- 色差评级

### 3.3 印花工艺流程

```
制版 → 调浆 → 印花 → 烘干 → 汽蒸 → 水洗 → 定型
```

### 3.4 染整工艺路线（可配置）

**典型工艺路线**：
```
预定型 → 染色 → 脱水 → 定型 → 检验
```

**多层级工艺路线**（好业财模式）：
- 主工艺路径下可嵌套预处理子流程、染色核心工序、后整理分支链路
- 每道工序绑定：特定设备组、工时定额、水电汽基准消耗值、关键控制点（如 pH 值区间、温度曲线拐点）
- 不预设固定工序数量，企业可依实际产线配置灵活增删节点

### 3.5 工艺参数与质量双向映射

当某缸布匹在验布环节被判定为"色光偏黄"，系统可自动反向锁定：
- 该缸所用染料批次
- 助剂添加顺序
- 蒸化温度记录
- 当日蒸汽压力波动曲线

形成完整的工艺溯源图谱，工程师可在统一界面完成根因分析与工艺优化建议生成。

**检验流程**：
- 检验员在验布机上扫描条码
- 输入疵点类型（断经、油污、色花等）
- 系统自动计算匹分、等级
- 生成检验报告

**面料检验十项指标**：
1. 色差
2. 纬斜
3. 缩水率
4. 克重偏差
5. 门幅偏差
6. 色牢度
7. 疵点
8. 强力
9. 起毛起球
10. 手感

**质检结果分级**：
- A 级（合格）
- B 级（让步接收，降级销售）
- C 级（不合格，返工或报废）

**疵点评分规则**：
- 四分制
- 十分制

## 五、面料行业成本核算体系

### 5.1 成本构成（四大类）

| 成本类别 | 占比 | 内容 |
|----------|------|------|
| 直接材料 | 60%-75% | 主要原料（棉/化纤/染料/助剂）+ 辅助材料 + 动力能源 + 边角料/损耗处理 |
| 直接人工 | 10%-18% | 生产工人工资 + 人工附加费用（社保/公积金/福利/加班费） |
| 制造费用 | 12%-20% | 设备折旧/维修/保养 + 车间间接费用 + 其他间接生产费用（环保处理费/质检费） |
| 外协加工费 | 视情况 | 委外纺纱/织布/印染/后整理费用 + 运费 + 辅料垫付 |

### 5.2 成本核算对象

1. **按产品品类**：纯棉纱/混纺纱/化纤纱/纯棉布/混纺布/色织布/印花布/染色布
2. **按生产订单/批次**：按客户订单、生产投料批次核算（**纺织行业主流核算方式**）
3. **按生产工序**：纺纱半成品/织布半成品/印染半成品/成品面料
4. **按生产车间**：纺纱车间/织布车间/印染车间/后整理车间

### 5.3 成本核算方法

**推荐：品种法 + 分批法 + 分步法结合**
- 大批量标准化生产 → 品种法
- 小批量定制生产 → 分批法
- 多工序连续生产 → 分步法

**逐步结转法 + 定额比例法**（坯布 + 染费模式）：

```
胚布阶段：算自产半成品成本 = 料 + 工 + 费 → 转入 半成品-胚布
染色阶段：胚布 + 外发染费 → 转入 库存商品-成品布
```

### 5.4 委托加工物资核算（关键流程）

**业务场景**：发出 300 吨自制半成品-胚布（成本 50 万）去染色，回来后收回 298 吨成品（正常损耗 2 吨），染费 10 万元

**会计分录**：

```
1. 发出胚布外发加工
   借：委托加工物资    500,000
   贷：自制半成品-胚布  500,000

2. 支付染费
   借：委托加工物资    100,000
       应交税费-进项税额  13,000
   贷：银行存款        113,000

3. 完工入库（合理损耗只影响单位成本，不影响总成本）
   入库成本 = 500,000 + 100,000 = 600,000 元（由 298 吨承担）
   借：库存商品-成品布  600,000
   贷：委托加工物资    600,000
```

**关键规则**：
- 正常损耗：直接摊入委托加工物资成本，最后进库存商品成本（不单独做分录，按实际收回数量结转）
- 非正常损耗（丢失/人为损坏）：计入营业外支出/管理费用，不能进成本

### 5.5 完整生产流程成本核算

**示例：纺织企业 A 生产棉纱 + 委托织布 + 销售成衣**

```
1. 采购原材料
   借：原材料-棉纱    1,020,000（100万+2万运费）
       应交税费-进项税  131,800
   贷：银行存款      1,151,800

2. 领用原材料
   借：生产成本-纺纱车间（直接材料）  800,000
```

### 5.6 月末成本单价计算

**公式**：
```
月末成本单价 = （当月领用胚布采购成本 + 对应加工费） ÷ 当月完工面料总数量
```

未完工的暂挂"生产成本"科目。

## 七、计量单位换算体系

### 7.1 四大计量单位

| 单位 | 用途 | 说明 |
|------|------|------|
| 米（m） | 长度计量 | 主单位，按米计价 |
| 公斤（kg） | 重量计量 | 用于成本核算 |
| 码（yd） | 长度计量 | 出口贸易常用 |
| 匹 | 包装计量 | 整匹销售 |

### 7.2 换算公式

**长度换算**：
```
1 码 = 0.9144 米
1 米 = 1.0936 码
```

**重量换算**：
```
公斤数 = 米数 × 克重(g/m²) × 幅宽(m) ÷ 1000
```

**面积重量换算**：
```
1 oz/yd² = 33.9 g/m²
```

**匹与米换算**（示例）：
```
1 匹 = 100 米（可配置，不同面料不同）
```

### 7.3 弹力面料缩率处理

- 入库时记录"原长"
- 出库时自动换算"实际可用长度"
- 预设"缩水率参数"

### 7.4 零剪业务精度

- 小数位精确到 0.1 米
- 自动计算剩余面料长度

## 十一、化验室与技术管理真实业务流程

> 调研来源：苏州同凯印染 ERP、环思印染 ERP、中易染整 ERP、好业财印染 ERP（2026-07 真实业务资料）
> 补充调研：2026-07-15 WebSearch 真实业务资料（染厂化验室操作手册/印染厂技术部研发操作规程/印染学习与交流/福步外贸论坛跟单流程）
> 本章节为后续批次（423+）技术管理模块实现的强制依据，禁止臆造流程。

### 11.1 化验室打样完整流程（5 步闭环）

```
打样通知 → 打样（ABCD 多版样）→ 色样确认（OK 样）→ 复样 → 建数据库
   ↓            ↓                    ↓                  ↓           ↓
业务跟单      技术科               客户确认          打样车间     工艺/基础技术资料库
录入技术要求  记录 ABCD 多版小样   送样判定 OK       大货前复样   色号/配方库 OK 保存
在线打印      工艺处方             重打/意见         验证处方     可查/可调用
贴样送化验室  系统核算染色成本     进度可查          可量产性
```

**各步骤详细说明**：

1. **打样通知单（打色通知书）**
   - 业务跟单接到客户打样需求，录入规范的技术要求
   - **必填字段**（真实业务）：对色光源（D65/TL84/U3000/CWF 等多光源）、色牢度要求（皂洗/摩擦/日晒/耐氯/干洗）、染化料是否环保（Oeko-Tex/GOTS）、打样版数（通常 A/B/C/D 四版）、打样坯布及大小、打样坯布规格（纱支/成分/组织）、完成时间
   - **时间要求**（行业惯例）：染色烧杯样 3 天，印花样 10 天，色织样 10 天
   - 在线打印《打样通知单》，贴样送化验室
   - 系统自动生成工厂打样唯一色号
   - 打样任务自动提醒给打样间

2. **打样处方（ABCD 多版小样处方）**
   - 技术科接到打样通知，安排打样员
   - **ABCD 真实含义**（2026-07-15 调研修正）：ABCD 指**打样版数**，染色烧杯样通常打 A、B、C、D 四个样供客户选择，每个样可能是不同的染料组合、浓度梯度或工艺参数，而非处方分类
   - **来样分析**（开料前必做）：纤维成分判定（棉麻黏胶→活性/还原/硫化/直接染料；毛丝→酸性染料；涤纶→分散染料；锦纶→酸性/活性/直接/分散染料）、规格要求（线密度/纱支/色牢度/缩水率/吸湿排汗等整理）
   - **确定配方**：传统经验配色（参考档案处方）或计算机测配色系统预测（DATACOLOR/X-Rite，仅辅助不能替代人工）
   - **开料规范**：手工开料（称料误差 ±0.001g、溶解、移液、定容）或电脑开料机自动操作；母色浓度 1/200、2/200、4/200 三档；小于 0.15ml 需稀释 10/100/1000 倍
   - **染色工艺**：浸染（IR 红外小样机，染杯自动反转）或轧染
   - 系统根据染化料单价核算染色成本（为报价提供依据）
   - 可与 DATACOLOR 自动滴液设备连接，自动读取颜色配方

3. **色样确认（OK 样判定）**
   - 状态：待判定 / 已判定（OK / 重打）
   - **对色规范**（真实操作）：0/45 度观察（色样平放灯箱内，灯光垂直照射，眼睛 45 度观察）；客户特殊要求 45/0 度观察
   - **对色注意事项**：光源亮度充足、材质纹路同方向、背景色一致（白灰黑无色彩系列）、温度一致（出缸烘干需冷却）、避免残像影响（凝视 20 秒后互补色残留 30 秒）、光源固定/距离固定/角度固定/亮度固定
   - **色差控制**：色偏控制 4-5 级，不得低于 4 级；多光源需检查"跳灯"现象
   - 送样给客户，客户确认后录入意见（签字确认方可作为大货标准）
   - OK 样保存到色号/配方库
   - 打样跟踪：进度清晰，便于跟单查询

4. **复样（大货前验证）**
   - **真实定义**（2026-07-15 调研确认）：小样获客户确认后，进入车间生产前必须复样——用车间半制品布、按生产计划安排的染料助剂模拟大生产重现小样颜色
   - **复样要点**（真实业务）：
     - 复样必须采用车间准备试产的半制品布（不可用化验室存布）
     - 认真检查车间坯布批号、染料批号、助剂批号
     - 复样要按生产计划安排的纱种、浴比、染化料模拟大生产进行
     - 复样中如实完整记录及留样
     - 若发现色光不符应查明原因（坯布/染料/助剂/工艺差异）
     - 复样要落实审核制度，只有复样通过了的配方才能试大样
     - 复样重现性好是提高染色一次成功率的前提
   - **色差标准**：和客户确认样色差达到 4-5 级方可转入生产工序
   - **放样试产**（投量大或色光要求高的订单）：大批量生产前联合打样间与生产部放样，全程跟踪记录数据，掌握大小样色光偏向；放样颜色与小样差异较大时需再次放样
   - 验证处方可量产性（小样→大货的加成系数调整）
   - 复样通过后处方升级为大货处方模板

5. **建数据库（工艺/基础技术资料）**
   - 色号库管理（色号唯一编码）
   - 小样配方库（可查/可调用）
   - 复样配方库
   - 大货处方模板库
   - 通过关联自动调用优染色工艺，提升染色质量，减少回修次数

### 11.1.1 染色技术卡（研发输出物）

> 来源：印染厂技术部研发操作规程（2026-07-15 调研）

复样完成后由研发组长开《染色技术卡》，附三件套：
1. **配方表**：染料/助剂品种与用量、工艺参数（温度/时间/浴比/pH）
2. **核可样**：客户确认的 OK 样（小样）
3. **复色样**：车间复样产出的样品

染色技术卡流转：研发组长 → 染色课主管 → 车间主任/对色组长 → 设定染程 → 染色 → 附头缸样 → 归档

**头缸样管控**（真实业务关键）：
- 大生产后的每一缸出缸样，必须靠第一缸试产样（头缸样）
- **禁止**第二缸靠第一缸、第三缸靠第二缸的累积误差做法
- 老色号相隔一周再投产，必须提前先锋试产一缸
- 平缸与液流缸的染色处方不可通用，变更染色方式必须重新打样试产

### 11.2 大货处方与加料处方（生产执行核心）

**大货处方单流程**：
```
扫描流转卡条码 → 依据备布准确数量 → 加载小样处方或历史大货处方
  → 根据缸别的浴比/浴量 → 开具大货处方单 → 多联打印便于称料
```

**加料处方单流程**：
```
扫描流转卡条码 → 加载大货处方 → 登记需要加料的物料
  → 保存提交 → 系统自动生成《加料处方单》
```

**关键参数**：
- **浴比**：纱与水的重量比（如 1:5、1:8、1:2.84）
  - 实验室小样标准浴比 1:5
  - 车间大货常规浴比 1:8 以上
  - 低恒定浴比染色：前处理 1:4.34，染色 1:2.84
  - 浴比影响：染色稳定性、能耗、化学品耗用量
- **浴量**：染缸容积（升）
- **加成系数**：小样到大货的得色差异修正（染缸大小、纱重量不同导致浴比差异）

**处方开单管控目标**：
- 工艺处方开单效率、准确率高
- 便于查询历史处方
- 支持按时间段查询染化料耗用汇总报表

### 11.3 称料方案（两种模式）

1. **按单扣料**：处方开单时按扣料时间自动扣减染化料库存
2. **对接称料设备**：染料/助剂分别对接电子称，扫码称重确认投料

### 11.4 染化料主数据管理（好业财模式）

- 强制要求供应商提供符合 GB/T 20877《染料产品中 14 种有害芳香胺的限量及测定》技术文档
- 支持扫描二维码直连 MSDS 安全数据表
- 依据 GHS 分类自动标注危化品属性
- 联动仓库环境传感器触发温控告警
- 领用流程绑定工艺卡与缸号，PDA 扫码确认
- 同步采集电子称重数据与操作人员工号
- 用量分析：每米布耗用染料克重、每缸助剂配比偏差率

## 十二、车间生产执行真实业务流程

> 调研来源：同凯面料染色 ERP、环思印染 ERP、飞创纺织 ERP（2026-07 真实业务资料）
> 本章节为后续批次（423+）生产执行模块实现的强制依据。

### 12.1 流转卡（缸卡）条码管理

**核心机制**：缸号采用条码管理，各生产环节只需扫描缸卡上的条码即可完成相关操作。

**流转卡承载信息**：
- 缸号（唯一标识）
- 订单信息（客户/订单号/产品/色号）
- 染整要求与注意事项
- 工序路线
- 计划配布数量
- 条码（一维码/二维码）

**流转卡应用场景**：
```
白坯仓库：扫描缸号条码 → 自动出库
染色车间：扫描缸卡条码 → 输入生产进度（进缸/出缸状态）
称料室：  扫描流转卡条码 → 加载大货处方 → 称料
车间流转：扫描流转卡条码 → 登记工人 → 自动跟进工序和产量
成品入库：PDA 扫描卷唛条码 → 自动入库
发货：    输入或扫描缸号 → 获取缸号所有信息 → 发货
```

### 12.2 生产计划与排缸（3 步）

```
生产计划单 → 备布 → 排缸执行
   ↓            ↓          ↓
订单分缸/缸号计划  从坯布仓库领坯  可视化排缸
流转卡打印（含条码）备布数据采集    实际排缸
染整要求/注意事项  坯布自动出库    缸位分配
```

**生产计划单要素**：
- 类型：正常订单 / 回修订单
- 订单分缸/缸号计划（一缸排不下可灵活自由拆缸）
- 订单染整要求和注意事项
- 流转卡（缸卡）打印（含条码）

**备布**：
- 根据生产计划，从坯布仓库领坯备布生产
- 系统通过备布采集的数据进行坯布自动出库
- 同时为化验室开大货处方提供依据（准确数量）

**可视化排缸**：
- 图形化分卡排缸操作界面
- 自动智能分卡排缸
- 缸位生产监视器界面：直观显示每个缸位的当前生产缸号进度状态及待生产缸号信息
- 支持生产计划变更、缸变更、合缸、缸终止
- 缸优先级功能

### 12.3 车间工序流转

**核心机制**：根据实际车间布局和需要了解的关键工序，通过后台自定义流转工序。

**流转过程**：
```
物料在车间流转 → 扫描流转卡条码 → 登记工人
  → 系统自动跟进该缸布所处的工序 → 自动统计工人产量
```

**工序质量反馈单**：
- 工序质量问题反馈登记
- 包括处理意见和方式
- 异常情况、回修情况登记
- 各生产环节共同查找原因及处理办法

**车间报工终端**：
- 扫描工号、缸卡，带出染整流程作业信息
- 生产工序可追溯
- 工序异常统计

### 12.4 验布打卷与成品入库

**验布打卷流程**：
```
验布机对接码表/电子称 → 疵点采集 → 生成验布报告
  → 卷唛标签打印 → PDA 扫描卷唛条码 → 自动入库
```

**疵点采集**：
- 疵点类型：断经、油污、色花、破洞、纬斜等
- 系统自动计算匹分、等级
- 四分制 / 十分制评分

### 12.5 产量工资（计件计时）

**三维度产量统计**：
1. 工序产量（按工序统计）
2. 设备产量（按机台统计）
3. 工人产量工资（按工人统计）

**工价方案**：
- 工序工价定义
- 自动汇总进入财务工资核算模块
- 各车间生产环节的计件计时工资采集录入

### 12.6 能耗管理

- 时间段能耗登记（水/电/气）
- 单位能耗分析（每米布能耗、每缸能耗）
- 能耗产值分析
- 联动设备 IoT 接口实时采集

### 12.7 缸号全生命周期状态机

```
待排缸 → 已排缸 → 备布中 → 进缸染色 → 出缸 → 验布 → 入库 → 发货
   ↓        ↓         ↓          ↓         ↓       ↓      ↓      ↓
 可变更   可变更     坯布出库   进度采集   质检    分级    入库   应收
```

**状态变更操作**：
- 生产计划变更、缸变更、合缸、缸终止
- 缸优先级调整
- 回修订单重新进缸
