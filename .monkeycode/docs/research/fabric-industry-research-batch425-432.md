## 14.1 流转卡工序流转（批次 425）

### 14.1.1 流转卡的真实业务定义

**流转卡**（又称生产流程卡/工序流转卡/缸卡/工程卡）是印染车间贯穿生产全流程的**核心跟踪载体**，一卡对应一缸布的生产任务，承载从开卡到成品入库的全部工序信息。

**真实业务定义**（基于多家印染 ERP）：
- 流转卡 = 缸号 + 工单信息 + 工序路线 + 计划配布数量 + 条码
- 一张流转卡对应一个缸号（一缸布的生产任务）
- 流转卡采用条码管理，各生产环节只需扫描缸卡上的条码即可完成相关操作
- 流转卡是车间工序流转、产量统计、质量追溯、成本归集的共同索引

【来源：苏州同凯印染 ERP 方案 V2.0、广州德控 KESHTECH 染缸 ERP 操作说明书、中易染整 ERP】

### 14.1.3 扫码流转完整流程（PDA/工控终端）

基于 isoface 制程刷卡终端、思多系统 SPM、友控工控一体机的真实业务流程：

```
┌─────────────────────────────────────────────────────────────┐
│ 流转卡扫码流转完整链路（PDA/工控终端）                       │
└─────────────────────────────────────────────────────────────┘

1. 工单下发 → 流转卡打印（含条码）
   ↓
2. 工人到达工位，扫码终端扫描流转卡条码
   ↓ 系统识别工单号/缸号/工序路线
3. 系统自动带出：产品型号、工艺路线、作业指导书、当前工序
   ↓
4. 工人刷卡登记（签入）：记录工号、设备编号、开始时间
   ↓ 状态：待加工 → 加工中
5. 加工过程（可选）：录入工艺参数、疵点反馈、异常登记
   ↓
6. 工人扫码完工（签出）：记录结束时间、实际产量
   ↓ 状态：加工中 → 完工
7. 系统自动触发下一道工序的开工准备
   ↓ 状态：完工 → 转入下道
8. 重复 2-7 直至最后一道工序
   ↓
9. 完工入库：PDA 扫描卷唛条码 → 自动入库
   ↓ 状态：完工 → 完工入库
```

**真实业务约束**（基于 MES 扫码报工实施经验）：
- 条码建议采用 Code128 或 QR Code 格式，数据容量大且支持特殊字符
- 工单条码通常包含工单号、批次号、优先级等关键信息
- 部分工厂在条码中嵌入加密校验位，防止条码伪造和误扫
- 扫码终端硬件要求：防尘防水 IP54+、抗跌落 1.2 米+、支持 24 小时连续运行
- 扫码后系统记录：开始时间、操作员工号、设备编号，同时更新工序状态为"已完成"
- 数据实时性：从扫码到系统显示延迟不超过 3 秒
- 数据完整性：每笔报工都有时间戳和操作者记录，支持后续追溯

**车间报工终端真实操作**（同凯印染 ERP）：
- 扫描工号、缸卡，带出染整流程作业信息
- 生产工序可追溯
- 工序异常统计

### 14.1.4 工序间流转的状态机

基于真实业务提炼的流转卡工序状态机：

```
┌──────────────────────────────────────────────────────────┐
│ 流转卡工序状态机（单工序维度）                            │
└──────────────────────────────────────────────────────────┘

  待加工 ──签入──→ 加工中 ──签出──→ 完工 ──流转──→ 转入下道
     │                 │               │                  │
     │                 │               │                  ↓
     │                 │               │            完工入库（末道）
     │                 │               │
     ↓ 异常            ↓ 异常          ↓ 质检不合格
   暂停               暂停            回修（开内修卡）
     │                 │               │
     └─────────────────┴───────────────┴──→ 重新进缸/返修工序
```

**状态定义**：
| 状态 | 业务含义 | 触发动作 |
|------|----------|----------|
| 待加工 | 工序已排程，等待开工 | 上道完工自动转入 |
| 加工中 | 工序正在执行 | 工人扫码签入 |
| 完工 | 工序已完成 | 工人扫码签出 |
| 转入下道 | 已流转至下一道工序 | 系统自动触发 |
| 完工入库 | 最后一道工序完工并入库 | PDA 扫描卷唛条码 |
| 暂停 | 异常暂停（设备故障/缺料） | 人工登记异常 |
| 回修 | 质检不合格，开内修卡返修 | 质检不合格触发 |

**流转卡整体状态机**（缸号维度）：
```
开卡 → 待排缸 → 已排缸 → 备布中 → 进缸染色 → 出缸 → ... → 验布 → 入库 → 发货
                                                                ↓
                                                           回修（重新进缸）
```

【来源：综合同凯印染 ERP、KESHTECH、思多系统 SPM、isoface 制程刷卡终端】

### 14.1.5 流转卡与工单/缸号/批次的关系

```
销售订单（SalesOrder）
   ↓ 分解
生产工单（ProductionOrder）── 一张工单可分多个缸号
   ↓ 排缸
缸号（DyeLot）── 一缸排不下可拆缸（分卡）
   ↓ 生成
流转卡（FlowCard）── 一缸一卡（含条码）
   ↓ 流转
工序记录（ProcessLog）── 每道工序一条记录
   ↓ 完工
匹号/卷号（Piece）── 一缸布分多匹出布
```

**关键关系约束**：
- 一张流转卡 ↔ 一个缸号（1:1，除非拆卡/合卡）
- 一张工单 → 多个缸号（1:N，一缸排不下拆缸）
- 一个缸号 → 多个匹号（1:N，一缸布分多匹出布）
- 流转卡号唯一，回修卡号 = 原卡号 + A/B/C 后缀

### 14.1.6 分卡/合卡/拆卡业务场景

**分卡（分缸开卡/分匹开卡）**（KESHTECH 真实业务）：
- **场景**：机缸染色时，流程卡配布数量大于机缸所能染色的容量
- **操作**：将此缸的坯布分成两部分分别染色，重新打印流程卡
- **规则**：分卡后生成新的卡号，原卡号状态变更，新卡独立流转

**合缸**（同凯印染 ERP）：
- **场景**：多张小卡合并为一缸染色（订单量小，凑缸生产）
- **操作**：合缸后多张卡共享一个缸号，但保留各自卡号
- **规则**：合缸后染化料成本按布重比例分摊到各卡

**拆卡**（拆匹）：
- **场景**：一匹布过长，需拆分为多匹便于流转
- **操作**：拆卡后生成子卡号，关联母卡号
- **规则**：拆卡后的匹号独立，但溯源到母卡

**缸终止**（同凯印染 ERP）：
- **场景**：生产过程中因质量/工艺问题终止该缸生产
- **操作**：缸终止后状态变更，已投入成本归集到该缸

**缸变更/合缸/缸优先级**（同凯印染 ERP）：
- 提供生产计划变更、缸变更、合缸、缸终止等功能
- 提供缸优先级功能，及时、有效、灵活的指挥各个生产环节车间进行有序生产

【来源：广州德控 KESHTECH 染缸 ERP 操作说明书、中易染整 ERP、同凯印染 ERP】

### 14.1.8 现有项目实现差距

**现有实现**：
- `process_node.rs`：工艺节点实体，绑定 custom_order_id，含 node_type/node_name/sequence/status/operator_id
- `process_log.rs`：工艺日志，含 action/before_status/after_status/log_time

5. **缺少内修卡**：未实现回修卡号编码规则（原卡号+A/B/C 后缀）和质量异常登记→开内修卡流程

### 14.2.2 四分制疵点评分真实计算规则

**四分制评分标准**（AATCC 标准，基于 ASTM D5430 Four-Point System）：

| 疵点长度（英寸） | 疵点长度（厘米） | 扣分 |
|------------------|------------------|------|
| ≤ 3" | ≤ 7.5 cm | 1 分 |
| > 3", ≤ 6" | > 7.5 cm, ≤ 15 cm | 2 分 |
| > 6", ≤ 9" | > 15 cm, ≤ 22.5 cm | 3 分 |
| > 9" | > 22.5 cm | 4 分 |

**疵点评分原则**（真实业务约束）：
- A. 同一码（linear yard）中所有经、纬向的疵点扣分**不超过 4 分**
- B. 对于严重的疵点，每码疵点将被评为四分（如：无论直径大小，所有的洞眼、破洞都评为 4 分）
- C. 对于连续出现的疵点（横档、边至边色差、窄封或不规则布宽、折痕、染色不均匀等），每码疵点应被评为四分
- D. 布边 1 英寸内不扣分
- E. 任何大于针孔的洞均扣 4 分
- F. 无论经向或纬向，都以**看得见**为原则，按疵点评分给予正确扣分
- G. 除了特殊规定（如涂层上胶布），通常只检验布的正面

**疵点分值计算公式**：

```
公制公式（每 100 平方米扣分）：
单卷扣分 = (单卷布疵点总分 × 100) ÷ (单卷布长度(米) × 布匹有效宽度(米))
整批扣分 = (所有检验布卷的疵点总分 × 100) ÷ (所有布卷总长度(米) × 布匹有效宽度(米))

英制公式（每 100 平方码扣分）：
单卷扣分 = (单卷布疵点总分 × 3600) ÷ (单卷布长度(码) × 布匹有效宽度(英寸))
整批扣分 = (所有检验布卷的疵点总分 × 3600) ÷ (所有布匹总长度(码) × 布匹有效宽度(英寸))
```

**计算示例**：
- 单卷布长 100 米，有效幅宽 1.5 米，疵点总分 12 分
- 每 100 平方米扣分 = (12 × 100) ÷ (100 × 1.5) = 8 分/100㎡

**不同布种的接受水平**（AATCC 标准，单卷/整批）：

| 布种类型 | 单卷（分/100 平方码） | 整批（分/100 平方码） |
|----------|----------------------|----------------------|
| 梭织-全人造/聚酯/尼龙/醋酸/衬衫衣料/精纺毛料 | 20 | 16 |
| 梭织-粗斜纹棉布/帆布/府绸/牛津/毛织品 | 28 | 20 |
| 梭织-亚麻布/薄细棉布 | 40 | 32 |
| 梭织-多皮奥尼丝绸/轻丝绸 | 50 | 40 |
| 针织-全人造/聚酯/尼龙/人造丝/精纺毛料 | 20 | 16 |
| 针织-全专业布匹/提花/灯心绒/丝绒 | 25 | 20 |
| 针织-基本针织（精梳棉/混纺棉） | 30 | 25 |
| 针织-基本针织（经梳毛机梳理过的棉布） | 40 | 32 |

**拒收标准**：
- 单卷布疵点评分超过指定分数 → 二等品
- 整批布匹的平均评分超过指定分数 → 该批布匹未通过检验
- 疵点连续出现下三米或以上 → 不论疵点评分多少，拒收

【来源：AATCC 检验及抽样标准、面料检验四分制与十分制异同点、deli-qc 面料外观检验方法】

### 14.2.3 十分制疵点评分真实计算规则

**十分制评分标准**（中国验布标准，基于十分制原理）：

**经向疵点扣分法**：
| 疵点长度 | 扣分 |
|----------|------|
| 1" 以下 | 1 分 |
| 1" - 5" | 3 分 |
| 5" - 10" | 5 分 |
| 10" - 36" | 10 分 |

**纬向疵点扣分法**：
| 疵点长度 | 扣分 |
|----------|------|
| 1" 以下 | 1 分 |
| 1" - 5" | 3 分 |
| 5" - 半门幅 | 5 分 |
| 半门幅以上 | 10 分 |

**十分制评分原则**：
- A. 同一码中所有经、纬向的疵点扣分**不超过 10 分**
- B. 破洞不问大小扣 10 分
- C. 布边**半英寸**内不扣分（注意：四分制是 1 英寸）
- D. 连续性疵点须开裁或降等外品
- E. 任何大于针孔的洞均扣 10 分

**等级计算方法**：
- **可接受范围** = 总分数 < 总码数（即平均每码扣分 < 1 分）
- **百码扣分不得超过 100 分**
- 若织物总罚分**少于**总码数 → **一级布（First Quality）**
- 若织物总罚分**超过**总码数 → **次级布（Second Quality）**
- 布宽 50 寸以上者，总罚分可酌量增加至不超过 10% 作为评定一级布之基准

【来源：中国验布标准、AATCC 检验及抽样标准、面料检验四分制与十分制异同点】

### 14.2.4 打卷（卷布/成卷）业务流程

**打卷流程**：
```
验布完成 → 系统自动按码表采集长度 → 按匹分码 → 打卷
   ↓                                            ↓
合格匹                                    疵点超标匹
   ↓                                            ↓
贴成品标签（含卷号/匹号/缸号/长度/重量/等级）   隔离返工
   ↓
PDA 扫描卷唛条码 → 自动入库
```

**卷唛标签核心字段**：
- 卷号/匹号（唯一标识）
- 缸号（关联染色批次）
- 色号、色名
- 长度（米/码）、重量（公斤）
- 幅宽、克重
- 等级（A/B/C 或 一级/次级）
- 条码（一维码/二维码，用于 PDA 扫描）
- 生产日期、操作员

### 14.2.5 匹号/卷号生成规则

基于现有项目 `inventory_piece.rs` 和真实业务：

**匹号生成规则**：
- 匹号在缸号范围内唯一（UNIQUE(dye_lot_no, piece_no)）
- 编码示例：`缸号-序号`（如 `DL20260715001-01`）
- 母卷拆分时生成子卷号，关联 parent_piece_id

**卷号生成规则**：
- 卷号 = 匹号 + 卷序（一匹可分多卷）
- 每卷独立条码，支持 PDA 扫码入库/出库

### 14.2.6 验布记录与质量分级（A/B/C）的关联

**质量分级规则**（基于四分制/十分制评分）：

| 等级 | 四分制标准（分/100 平方码） | 十分制标准 | 业务处理 |
|------|----------------------------|------------|----------|
| A 级（一级品） | ≤ 单卷接受水平（如 20/28/40/50） | 总罚分 < 总码数 | 全额计件工资，正常销售 |
| B 级（二级品） | 超过单卷接受水平但可降价销售 | 总罚分略超总码数 | 折扣计件工资，降价销售 |
| C 级（等外品） | 严重超标或连续疵点 | 总罚分远超总码数 | 不计件工资，开裁或降等外品 |

**屡犯疵点判定**：
- 任何重复或不断出现的疵点都将构成屡犯的疵点，对每码布匹出现的屡犯的疵点都必须处以四分
- 无论疵点分数是多少，任何有十码以上布匹含有屡犯的疵点的卷，都应当被定为不合格

**全幅宽度疵点**：
- 每 100 平方码内含有多于四处全宽疵点的卷，不得被评定为一等品
- 平均每 10 个直线码数内含有一个以上重大疵点的卷将被定为不合格
- 在布头或布尾 3 码内含有一个重大的疵点的卷都应评定为不合格

【来源：AATCC 检验及抽样标准、面料检验四分制与十分制异同点】

### 14.3.1 计件工资的真实业务（按米/按公斤/按缸）

2. **设备产量**（按机台统计）：每台设备的加工产量
3. **工人产量工资**（按工人统计）：每个工人的计件工资

**计件维度**（印染行业真实业务）：
- **按米计件**：验布、定型、卷布等按长度工序

**产量数据来源**：

- 辅以"日产量报表"
- 扫码报工自动采集（工序签入签出记录产量）

### 14.3.2 不同工序的计件单价规则

**计件单价确定方法**（基于真实业务资料）：

**方法一：标准工时法**（IE 工业工程法）
```
计件单价 = 小时工资标准 × 标准工时（小时）
示例：小时工资 28.7 元，标准工时 2.2 分钟（0.0367 小时）
单价 = 28.7 × 0.0367 ≈ 1.05 元/件
```

**方法二：岗位工资基数法**（2025 计件工资核算管理细则）
```
计件单价 = （岗位工资基数 × 技能系数） ÷ （月标准制度工时 × 单位定额工时）
- 岗位工资基数：根据岗位评价结果确定
- 技能系数：初级 1.0、中级 1.1、高级 1.2、技师 1.3
- 月标准制度工时：21.75 天 × 8 小时 = 174 小时
```

**方法三：等级分钟值法**（制衣/印染工序表应用）
```
工价 = 等级分钟值 × 标准工时
等级分钟值举例：
  A 级 0.35 元/分钟
  B 级 0.30 元/分钟
  C 级 0.25 元/分钟
  D 级 0.20 元/分钟
  E 级 0.15 元/分钟
```

**不同工序的计件单价规则**（印染行业典型）：

| 工序 | 计件单位 | 单价影响因素 | 典型单价范围 |
|------|----------|--------------|--------------|
| 染色工 | 按缸/按公斤 | 缸型（大/中/小）、颜色深浅、工艺复杂度 | 50-200 元/缸 |
| 印花工 | 按米/按套 | 花型复杂度、套色数、布种 | 0.1-0.5 元/米 |
| 定型工 | 按米 | 幅宽、克重、工艺参数 | 0.05-0.2 元/米 |
| 验布工 | 按米 | 布种、检验标准 | 0.03-0.1 元/米 |
| 质检员 | 按米/按匹 | 检验项目数量 | 0.05-0.15 元/米 |
| 仓管员 | 按件/按吨 | 出入库操作类型 | 计时为主 |

### 14.3.3 A/B/C 级品的计件折扣规则

**质量系数设计**（基于真实业务资料）：

```
实际工资 = 合格产品数量 × 计件单价 × 质量系数 + 其他激励
```

**A/B/C 级品折扣规则**（印染行业典型）：

| 等级 | 质量系数 | 业务含义 | 工资影响 |
|------|----------|----------|----------|
| A 级 | 1.0（全额） | 一次染色合格，质量优秀 | 全额计件 |
| B 级 | 0.8（8 折） | 轻微疵点，可降价销售 | 折扣计件 |
| C 级 | 0.5（5 折）或不计 | 严重疵点，等外品 | 折扣或不计件 |

**质量基准线设计**（通用制造业实践）：
- 合格率 ≥ 98%：质量系数 = 1.0（拿全额）
- 合格率 95%-98%：质量系数 = 0.8（打 8 折）
- 合格率 < 95%：质量系数 = 0.5（打 5 折），且承担部分返工成本

**返工成本承担**（真实业务约束）：
- 因人为原因造成批量质量返工或报废的，需承担相应的经济赔偿责任
- 返工成本示例：800 件 × 返工成本 10 元/件 = 8000 元，员工承担 30% = 2400 元

**追溯机制**：
- 每件产品贴员工工号或批次码
- 出现质量问题，能追溯到具体员工
- 避免"大锅饭"，谁的责任谁承担

【来源：制造业财务计件工资设计、2025 计件工资核算及发放管理细则、南泉制衣工序表应用】

### 14.3.4 班组计件 vs 个人计件

**个人计件**：
- 适用：独立操作的工序（如验布、卷布）
- 计算：个人产量 × 单价 × 质量系数
- 数据来源：扫码报工记录操作员工号

**班组计件**：
- 适用：需要协作的工序（如染色、定型）
- 计算：
  ```
  班组总工资 = 班组总产量 × 单价 × 质量系数
  个人工资 = 班组总工资 × 个人分配系数
  ```
- 分配方式（推荐）：
  - 产量占比 40%
  - 质量占比 40%
  - 协作占比 20%

**团队奖励**：
- 对需要协作的工序，按团队总产量计算总工资
- 再按个人贡献分配（产量占比 40% + 质量占比 40% + 协作占比 20%）
- 避免"个人抢活、团队内耗"

```
   ↓

   ↓

   ↓

```

- 各车间生产环节的计件计时工资采集录入

### 14.4.3 机台效率分析

**OEE（设备综合效率）分析**：
```
OEE = 时间开动率 × 性能开动率 × 合格品率
时间开动率 = 实际运行时间 / 计划运行时间
性能开动率 = 实际产量 / 理论产量
合格品率 = 合格品数量 / 总产量
```

**机台效率分析维度**：
- 设备开动时间、停机时间、故障时间
- 设备产能利用率
- 单位时间产量（米/小时、公斤/小时）
- 能耗产值（单位能耗的产量）

### 14.5.2 染料分类的真实业务区别

**染料分类**（基于纤维亲和力分类）：

| 染料类型 | 适用纤维 | 染色机理 | 典型应用 |
|----------|----------|----------|----------|
| **活性染料** | 棉、麻、黏胶等纤维素纤维 | 与纤维发生共价键结合 | 棉布染色主力，色泽鲜艳，色牢度好 |
| **分散染料** | 涤纶、锦纶等合成纤维 | 染料分散到纤维内部 | 涤纶染色主力，高温高压染色 |
| **还原染料** | 棉、麻等纤维素纤维 | 还原后染色再氧化 | 高级色牢度要求，但工艺复杂 |
| **直接染料** | 棉、麻、黏胶 | 直接上染纤维 | 工艺简单，但色牢度较差 |
| **酸性染料** | 羊毛、蚕丝、锦纶 | 酸性条件下染蛋白质纤维 | 毛丝染色主力 |
| **阳离子染料** | 腈纶 | 阳离子与纤维阴离子结合 | 腈纶染色主力 |
| **硫化染料** | 棉 | 硫化还原染色 | 深色棉布，成本低 |

**真实业务约束**：
- 不同纤维需选择对应染料类型（来样分析必做）：
  - 棉麻黏胶 → 活性/还原/硫化/直接染料
  - 毛丝 → 酸性染料
  - 涤纶 → 分散染料
  - 锦纶 → 酸性/活性/直接/分散染料
- **平缸与液流缸的染色处方不可通用**，变更染色方式必须重新打样试产

【来源：好业财印染 ERP、chemball 化工产品主数据、印染厂技术部研发操作规程】

### 14.5.4 染化料与配方的关联

**染化料与染色配方的关系**：
```
染化料主数据（DyeChemical）
   ↓ 被引用
染色配方（DyeRecipe）
   ├── 染料明细（染料编码 + 浓度% + 用量）
   └── 助剂明细（助剂编码 + 浓度% + 用量）
   ↓ 被调用

   ├── 工单号/流转卡号
   ├── 缸号、浴比、浴量

```

**用量计算公式**：
```
用量 = 浓度(%) × 布重(kg) × 浴比 / 100
或按 o.w.f（on weight of fabric）百分比直接计算
```

### 14.5.5 效期管理（活性染料保质期预警）

**效期管理真实业务**：
- 分散剂 NNO：有效期一年，超期需经检验合格后方可使用
- 分散剂 MF：保质期 12 个月（密封干燥存放）
- 十八胺聚氧乙烯醚 AC-1815：保质期 12 个月
- 活性染料：保质期通常 1-2 年，存储条件要求高（防潮/避光）

**效期预警机制**：
- 入库时记录生产日期、保质期
- 系统自动计算有效期至
- 接近有效期（如 30/60/90 天）触发预警
- 超期染料检验合格后方可使用，否则报废

- 分散剂 NNO：25KG/塑编袋 或 40KG/袋
- 分散剂 MF：25kg/袋
- 染料：25kg/桶 或 50kg/桶

## 14.6 委托加工物资（批次 430）

### 14.6.1 委托加工的真实业务全流程

基于企业会计准则、会计百科、做账实操的真实业务：

```
┌─────────────────────────────────────────────────────────────┐
│ 委托加工物资完整业务流程                                      │
└─────────────────────────────────────────────────────────────┘

1. 签订委托加工合同（明确允许损耗率）
   ↓
2. 发出原材料/半成品
   借：委托加工物资
       贷：原材料/自制半成品
   ↓
3. 支付加工费、运杂费
   借：委托加工物资
       应交税费——应交增值税（进项税额）
       贷：银行存款
   ↓
4. 加工过程（受托方加工，可监工）
   ↓
5. 收回加工物资（实际收回数量可能 < 发出数量）
   ↓
   ├─ 正常损耗（工艺允许、自然损耗）→ 摊入成本
   └─ 非正常损耗（管理不善、意外事故）→ 营业外支出
   ↓
6. 入库
   借：原材料/库存商品
       贷：委托加工物资
   ↓
7. 对外销售结转成本
   借：主营业务成本
       贷：库存商品
```

### 14.6.2 委托加工单据字段

**委托加工合同字段**：
- 合同号、委托方、受托方
- 加工物料（品名/规格/数量）
- 允许损耗率（关键，需在合同中明确）
- 加工费单价、总价
- 交货期、运输方式
- 结算方式

**委托加工发出单字段**：
- 单据号、日期
- 委托加工合同号
- 发出物料编码、名称、规格
- 发出数量、单位成本、总成本
- 受托方、预计收回日期

**委托加工收回单字段**：
- 单据号、日期
- 委托加工发出单号
- 收回物料编码、名称、规格
- 收回数量（实际）
- 合格数量、损耗数量
- 损耗类型（正常/非正常）
- 加工费、运杂费
- 入库仓库、库位

### 14.6.3 正常损耗 vs 非正常损耗的处理规则（关键）

**核心规则**（基于企业会计准则，关键业务约束）：

| 损耗类型 | 定义 | 会计处理 | 进项税处理 |
|----------|------|----------|------------|
| **正常损耗** | 工艺允许、自然损耗（如机械加工 3%-5%、食品加工 5%-8%、印染 4%-6%） | 摊入委托加工物资成本（通过提高单位成本分摊至合格物资） | 进项税**无需转出** |
| **非正常损耗** | 管理不善、意外事故（如火灾、被盗、丢失） | 计入当期损益（营业外支出/管理费用），**不能进成本** | 进项税**必须转出**（损耗成本 × 适用税率） |

**正常损耗会计处理**（关键：不单独做分录，按实际收回数量结转）：

```
场景：发出 1000 件原材料成本 100 万，允许损耗率 ≤ 3%（30 件）
      支付加工费 10 万（含税 11.3 万，税率 13%）
      实际收回 970 件（正常损耗 30 件）

1. 发出原材料
   借：委托加工物资—XX 加工厂  1,000,000
       贷：原材料—XX 材料  1,000,000

2. 支付加工费
   借：委托加工物资—XX 加工厂  100,000
       应交税费——应交增值税（进项税额）  13,000
       贷：银行存款  113,000

3. 加工完成收回物资（正常损耗通过提高单位成本分摊）
   总成本 = 1,000,000 + 100,000 = 1,100,000 元
   单位成本 = 1,100,000 ÷ 970 ≈ 1,134.02 元/件
   借：原材料—加工后材料  1,100,000
       贷：委托加工物资—XX 加工厂  1,100,000
```

**非正常损耗会计处理**（关键：通过"待处理财产损溢"过渡，进项税转出）：

```
场景：发出 1000 件原材料成本 100 万，丢失 50 件（非正常损耗）
      单价 1000 元/件，损耗成本 = 50 × 1000 = 50,000 元
      税率 13%，进项税转出 = 50,000 × 13% = 6,500 元

1. 发出原材料
   借：委托加工物资—XX 加工厂  1,000,000
       贷：原材料—XX 材料  1,000,000

2. 确认丢失损耗成本
   借：待处理财产损溢—待处理流动资产损溢  50,000
       贷：委托加工物资—XX 加工厂  50,000

3. 查明原因（管理不善，责任人赔偿 20%）
   进项税转出 = 50,000 × 13% = 6,500 元
   借：管理费用—存货损失  46,500
       其他应收款—责任人  10,000
       贷：待处理财产损溢—待处理流动资产损溢  50,000
       应交税费——应交增值税（进项税额转出）  6,500
```

**关键注意事项**（真实业务约束）：
1. **损耗率界定**：需在《委托加工合同》中明确"允许损耗率"，无约定时按行业标准执行；超标准损耗需取得加工厂书面说明，否则税务机关可能认定为非正常损耗
2. **进项税额处理**：正常损耗的进项税额无需转出；非正常损耗需按损耗成本 × 适用税率转出
3. **凭证留存**：非正常损耗需留存"损耗证明、责任认定、报警记录（意外事故）"；正常损耗需留存"损耗确认单、行业损耗标准说明"
4. **成本分摊**：正常损耗通过"单位成本=总成本÷合格数量"分摊，**不得直接冲减原材料成本**，避免成本核算失真

【来源：会计百科委托加工物资成本、做账实操-委托加工损耗会计分录 SOP、找法网委托加工科目、初级会计存货】

### 14.6.4 委托加工费核算

**委托加工物资成本构成**：
```
委托加工物资成本 = 实际耗用物资成本 + 加工费 + 运杂费 + 装卸费 + 相关税费
```

**不能计入委托加工物资成本的项目**：
1. 委托方为一般纳税人，受托方代扣增值税（可抵扣）→ 不能计入成本
2. 非正常消耗的直接材料、直接人工和制造费用 → 计入当期损益"营业外支出"
3. 企业商品入库后的仓储费用 → 计入当期损益（但委托加工过程中必要的仓储费用计入成本）
4. 为达到营销目的样品 → 计入销售费用

### 14.6.5 与库存/成本/凭证的关联

```
委托加工发出 → 库存减少（原材料/半成品出库）
   ↓
委托加工物资科目归集成本
   ↓
委托加工收回 → 库存增加（原材料/库存商品入库）
   ↓
成本核算 → 委托加工物资成本转入库存商品成本
   ↓
凭证生成 → 自动生成会计凭证
   ├── 发出：借委托加工物资/贷原材料
   ├── 加工费：借委托加工物资/应交税费/贷银行存款
   ├── 非正常损耗：借待处理财产损溢/贷委托加工物资
   ├── 损耗处理：借管理费用/其他应收款/贷待处理财产损溢/应交税费
   └── 入库：借库存商品/贷委托加工物资
```

## 14.7 多业务模式（批次 431）

|------|----------|----------|----------|----------|
| **模式 1：坯布经销（采购坯布→销售）** | 采购坯布→库存→销售 | 自有 | 商品销售收入 | 采购成本 |
| **模式 2：坯布经销（库存坯布→销售）** | 库存坯布→销售 | 自有 | 商品销售收入 | 加权平均成本 |
| **模式 3：成品经销（采购坯布→染整→销售）** | 采购坯布→染整→销售 | 自有 | 商品销售收入 | 坯布成本+加工费 |
| **模式 4：成品经销（库存坯布→染整→销售）** | 库存坯布→染整→销售 | 自有 | 商品销售收入 | 坯布成本+加工费 |
| **模式 5：染整加工（客供坯布→本厂染整）** | 客供坯布→本厂染整 | 客户所有 | 加工费收入 | 加工成本 |
| **模式 6：染整加工（客供坯布→本厂/委外加工）** | 客供坯布→本厂/委外加工 | 客户所有 | 加工费收入 | 加工成本+委外费 |

**补充业务模式**（基于行业实践）：

| 模式 | 业务流程 | 物料归属 | 收入类型 |
|------|----------|----------|----------|
| **自织自染** | 采购原料→纺纱→织布→染整→销售 | 自有 | 商品销售收入 |
| **来料加工** | 客户来料→加工→收取加工费 | 客户所有 | 加工费收入 |
| **委托加工（发出）** | 自制半成品→委外加工→收回成品→销售 | 自有（发出+收回） | 商品销售收入 |

### 14.7.2 不同业务模式的单据流转差异

**模式 1/2：坯布经销**（纯贸易）
```
采购订单 → 采购入库单 → 库存 → 销售订单 → 销售出库单 → 应收
   ↓                                            ↓
   应付                                      结转成本
```

**模式 3/4：成品经销**（经销+染整）
```
采购订单（坯布） → 采购入库单 → 生产订单（染整）→ 完工入库单 → 销售订单 → 销售出库单 → 应收
   ↓                                ↓                                    ↓
   应付                         染化料领用                              结转成本
                                加工费（如委外）
                                能耗/人工归集
```

**模式 5/6：染整加工**（客供坯布）
```
客供坯布入库（备查，不入正式科目）→ 生产订单（染整）→ 完工交货 → 加工费结算 → 应收
                                       ↓
                                  染化料领用（自有）
                                  能耗/人工归集
                                  加工成本核算
```

**自织自染**（全产业链）
```
采购原料 → 纺纱工单 → 织布工单 → 染整工单 → 完工入库 → 销售订单 → 销售出库 → 应收
   ↓           ↓           ↓           ↓                            ↓
   应付     半成品入库   半成品入库   染化料领用                   结转成本
```

### 14.7.3 不同业务模式的成本核算差异

| 业务模式 | 成本核算对象 | 成本构成 | 核算方法 |
|----------|--------------|----------|----------|
| 坯布经销 | 坯布 SKU | 采购成本+运费 | 加权平均/先进先出 |
| 成品经销 | 成品布+缸号 | 坯布成本+染化料+加工费+能耗+人工 | 按缸号实际成本法 |
| 染整加工 | 加工订单+缸号 | 染化料+能耗+人工（不含坯布） | 加工成本法 |
| 自织自染 | 成品布+缸号 | 原料+纺纱成本+织布成本+染整成本 | 分步法（半成品→成品） |
| 委托加工 | 加工物资+缸号 | 发出物资成本+加工费+运费 | 委托加工物资核算 |
| 来料加工 | 加工订单 | 加工成本（不含来料） | 加工成本法 |

**关键差异**：
- 经销模式：成本含坯布
- 加工模式：成本**不含**客供坯布（仅加工成本）
- 自织自染：成本含全产业链（原料+纺纱+织布+染整）
- 委托加工：成本含发出物资+加工费

### 14.7.4 不同业务模式的收入确认差异

| 业务模式 | 收入类型 | 收入确认时点 | 收入金额 |
|----------|----------|--------------|----------|
| 坯布经销 | 商品销售收入 | 坯布交付 | 坯布销售价 |
| 成品经销 | 商品销售收入 | 成品交付 | 成品销售价 |
| 染整加工 | 加工费收入 | 加工完成交付 | 加工费（不含坯布价值） |
| 自织自染 | 商品销售收入 | 成品交付 | 成品销售价 |
| 委托加工（发出方） | 商品销售收入 | 收回成品销售时 | 成品销售价 |
| 来料加工 | 加工费收入 | 加工完成交付 | 加工费（不含来料价值） |

**关键差异**：
- 经销模式：全额确认收入（含坯布价值）
- 加工模式：仅确认加工费收入（**不含**客供/来料价值）
- 受托方收到委托材料时：**仅做备查登记，不进正式会计科目**（因材料所有权属委托方）

## 14.8 缸号状态机（批次 432）

### 14.8.1 缸号完整生命周期

基于同凯印染 ERP、中易染整 ERP、印染作业管理手册的真实业务：

```
┌─────────────────────────────────────────────────────────────┐
│ 缸号完整生命周期（10 道工序）                                 │
└─────────────────────────────────────────────────────────────┘

1. 投缸（备布进缸）
   ↓ 状态：进缸染色
2. 染色（核心上色工序）
   ↓ 升温梯度保温，每缸中途取样对色
3. 出缸对色（第一道色检）
   ↓ 色差合格流转；超标退回重新调色复染
4. 皂洗（去除浮色）
   ↓ 95℃热水煮碱，去除染料浮色与残余杂质
5. 固色（提高色牢度）
   ↓ 无醛固色剂提高染料固着率
6. 脱水（去除水分）
   ↓
7. 烘干（130℃，超喂 100%）
   ↓ 烘干至布面完全干燥
8. 验布（全外观质检）
   ↓ 验布机逐匹检查疵点
9. 入库（PDA 扫描卷唛条码）
   ↓ 状态：完工入库
10. 发货（扫描缸号条码）
   ↓ 状态：已发货
```

### 14.8.2 缸号状态机的所有状态和流转规则

**缸号状态机**（基于真实业务提炼）：

```
┌──────────────────────────────────────────────────────────────────┐
│ 缸号状态机完整流转图                                              │
└──────────────────────────────────────────────────────────────────┘

  待排缸 ──排缸──→ 已排缸 ──备布──→ 备布中 ──进缸──→ 进缸染色
     │                │               │                    │
     │ 可变更         │ 可变更        │ 坯布出库           │ 进度采集
     │                │               │                    ↓
     │                │               │              皂洗/固色/脱水/烘干
     │                │               │                    │
     │                │               │                    ↓
     │                │               │              验布（质检）
     │                │               │                    │
     │                │               │                    ↓
     │                │               │              入库（完工）
     │                │               │                    │
     │                │               │                    ↓
     │                │               │              发货（已发货）
     │                │               │
     │                │               │
     ↓                ↓               ↓
   可变更           可变更         坯布出库
                                              质检不合格
                                                   ↓
                                              回修（重新进缸）
                                                   ↓
                                              回修状态（原卡号+A/B/C）
```

**状态定义**：

| 状态 | 业务含义 | 触发动作 | 可变更操作 |
|------|----------|----------|------------|
| 待排缸 | 缸号已生成，等待排缸 | 订单分缸生成缸号 | 可变更、可取消 |
| 已排缸 | 已分配染缸位 | 可视化排缸 | 可变更、合缸、缸终止 |
| 备布中 | 从坯布仓库领坯备布 | 备布采集 | 坯布自动出库 |
| 进缸染色 | 投缸染色 | 扫描缸卡条码 | 进度采集（进缸/出缸状态） |
| 皂洗 | 皂洗工序 | 工序流转 | - |
| 固色 | 固色工序 | 工序流转 | - |
| 脱水 | 脱水工序 | 工序流转 | - |
| 烘干 | 烘干工序 | 工序流转 | - |
| 验布 | 质检工序 | 工序流转 | 质检结果录入 |
| 入库 | 完工入库 | PDA 扫描卷唛条码 | 自动入库 |
| 发货 | 已发货 | 扫描缸号条码 | 生成应收 |
| 回修 | 质检不合格返修 | 开内修卡 | 重新进缸（原卡号+A/B/C） |
| 已取消 | 缸终止 | 缸终止操作 | - |

**关键流转规则**：
- 正向流转：待排缸 → 已排缸 → 备布中 → 进缸染色 → ... → 验布 → 入库 → 发货
- 回修流转：验布不合格 → 回修 → 重新进缸染色 → ... → 验布 → 入库
- 异常流转：任意非终态 → 已取消（缸终止）
- 回修卡号编码：原卡号 + A（一次回修）/ B（二次回修）/ C（三次回修）

### 14.8.3 缸号与工单/批次/质检/成本的关系

```
┌─────────────────────────────────────────────────────────────┐
│ 缸号关联关系图                                                │
└─────────────────────────────────────────────────────────────┘

销售订单（SalesOrder）
   ↓ 1:N 分解
生产工单（ProductionOrder）── 业务模式（经销/加工/委外）
   ↓ 1:N 排缸
缸号（DyeLot）── 缸号状态机
   ├── 1:N 匹号/卷号（Piece）── 验布打卷
   ├── 1:N 工序记录（ProcessLog）── 扫码报工
```

**关键关联约束**：

```
425 流转卡 ←─── 427 产量工资（扫码报工）
   │
   ├─── 426 验布打卷（流转卡+缸号）
   │
432 缸号状态机 ←── 428 能耗管理（按缸号归集）
   │
   ├─── 430 委托加工（缸号成本）
   │
429 染化料主数据 ←── 430 委托加工（染化料）
   │
431 多业务模式（贯穿全流程，依赖 425/432）
```

### 14.9.3 实现建议

1. **批次 425+432 优先实现**：流转卡和缸号状态机是面料行业 ERP 的两大基础，其他模块均依赖二者
2. **批次 426+429 并行实现**：验布打卷和染化料主数据相对独立，可并行
3. **批次 427+428 顺序实现**：产量工资依赖扫码报工（425），能耗管理依赖缸号（432）
4. **批次 430+431 最后实现**：委托加工和多业务模式涉及全流程集成，依赖前面批次
