## 三、路由配置总览

### 3.1 完整路由结构
```
/api/v1/erp/
├── /auth/                      # 认证管理（3 个接口）
├── /users/                     # 用户管理（5 个接口）
├── /roles/                     # 角色管理（7 个接口）
├── /products/                  # 产品管理（10 个接口）
│   ├── /                      # CRUD
│   ├── /batch/*               # 批量操作
│   └── /:product_id/colors/*  # 色号管理 ⭐
├── /product-categories/        # 产品类别（5 个接口）
├── /warehouses/                # 仓库管理（10 个接口）
│   ├── /                      # CRUD
│   └── /locations/*           # 库位管理 ⭐
├── /departments/               # 部门管理（6 个接口）
├── /dashboard/                 # 仪表板（4 个接口）
├── /finance/                   # 财务管理（10 个接口）
├── /sales/                     # 销售管理（11 个接口）
│   ├── /orders/*              # 通用订单
│   └── /fabric-orders/*       # 面料订单 ⭐
├── /inventory/                 # 库存管理（21 个接口）
│   ├── /stock/*               # 库存管理
│   ├── /stock/fabric          # 面料库存 ⭐
│   ├── /stock/transactions    # 库存流水 ⭐
│   ├── /stock/summary         # 库存汇总 ⭐
│   └── /transfers/*           # 调拨管理
├── /customers/                 # 客户管理（5 个接口）
├── /batches/                   # 批次管理（6 个接口）⭐
└── /gl/                        # 总账管理（9 个接口）⭐
```
