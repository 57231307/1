## 🔧 技术亮点

### 1. 事务处理
所有关键业务操作均使用数据库事务，确保数据一致性：
```rust
let txn = self.db.begin().await?;
// 多个数据库操作
txn.commit().await?;
```

### 2. 状态机模式
订单、盘点、收款等均实现完整状态流转：
```
draft → pending → approved → completed
         ↓
      rejected
```

### 3. 库存预留机制
支持订单审核时锁定库存，防止一货多卖：
```rust
// 锁定库存
create_reservation() → pending
lock_reservation() → locked
use_reservation() → used
```

### 4. 自动编号
所有业务单据均实现唯一编号生成：
- 销售订单：SO+ 日期 + 序号
- 盘点单：IC+ 日期 + 序号
- 收款单：PAY+ 日期 + 序号
- 客户编号：CUS+ 日期 + 序号

### 5. 差异计算
库存盘点自动计算盘盈盘亏：
```rust
quantity_variance = quantity_actual - quantity_book
```
