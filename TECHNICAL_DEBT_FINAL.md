## 核心算法实现

### 1. 直线法折旧算法
```rust
月折旧额 = (原值 - 净残值) / 使用年限（月）
累计折旧 = 月折旧额 × 已使用月数
净值 = 原值 - 累计折旧
```

### 2. 信用评估评分模型
```
总分 = 付款记录×30% + 合作时长×20% + 订单规模×25% + 信用记录×25%

信用等级：
- AAA (≥90 分): 50 万额度
- AA (≥80 分): 30 万额度
- A (≥70 分): 20 万额度
- BBB (≥60 分): 10 万额度
- BB (≥50 分): 5 万额度
- C (<50 分): 1 万额度
```
