## 三、技术方案详解

### 3.1 数据导入导出技术方案

```rust
// 导入导出框架核心设计
pub trait ImportExport<T> {
    async fn import(&self, data: &[u8], format: Format) -> Result<ImportResult, AppError>;
    async fn export(&self, query: &Query, format: Format) -> Result<Vec<u8>, AppError>;
    fn get_template(&self, format: Format) -> Result<Vec<u8>, AppError>;
}

pub struct ImportResult {
    pub total_count: usize,
    pub success_count: usize,
    pub error_count: usize,
    pub errors: Vec<ImportError>,
}
```

**关键设计决策**:
- 使用流式处理避免大文件内存溢出
- 错误数据单独报告，不影响其他数据导入
- 支持异步导入，通过WebSocket通知进度

### 3.2 消息通知中心技术方案

```rust
// 通知渠道抽象
#[async_trait]
pub trait NotificationChannel {
    async fn send(&self, message: &NotificationMessage) -> Result<(), AppError>;
    fn channel_type(&self) -> ChannelType;
}

// 通知服务
pub struct NotificationService {
    channels: HashMap<ChannelType, Box<dyn NotificationChannel>>,
    template_engine: Handlebars<'static>,
}
```

**关键设计决策**:
- 渠道可插拔，易于扩展新渠道
- 模板引擎使用Handlebars，支持条件逻辑
- 通知发送异步化，不影响主业务流程
- 失败通知进入死信队列，支持重试

### 3.3 工作流引擎技术方案

```rust
// 流程定义
pub struct WorkflowDefinition {
    pub id: i32,
    pub name: String,
    pub version: i32,
    pub nodes: Vec<WorkflowNode>,
    pub edges: Vec<WorkflowEdge>,
    pub start_node: i32,
    pub end_nodes: Vec<i32>,
}

// 流程引擎
pub struct WorkflowEngine {
    definitions: Arc<RwLock<HashMap<i32, WorkflowDefinition>>>,
    runtime: WorkflowRuntime,
}
```

**关键设计决策**:
- 流程定义与运行时分离，支持版本管理
- 状态机驱动流程流转
- 支持事件订阅，外部系统可监听流程状态

### 3.4 数据权限技术方案

```rust
// 字段权限
pub struct FieldPermission {
    pub field_name: String,
    pub permission: FieldAccess, // Read/Write/Hidden
}

// 数据范围
pub enum DataScope {
    Self,           // 仅本人
    Department,     // 本部门
    DepartmentTree, // 本部门及下属
    All,            // 全部
    Custom(String), // 自定义规则
}

// 权限上下文
pub struct PermissionContext {
    pub user_id: i32,
    pub role_ids: Vec<i32>,
    pub department_id: Option<i32>,
    pub data_scope: DataScope,
}
```

**关键设计决策**:
- 权限检查在Service层统一处理
- 数据范围通过SQL条件注入实现
- 字段权限通过DTO序列化时过滤实现
- 权限结果缓存，减少重复计算
