## 一、模块概述

### 1.1 面料行业财务核算特点

1. **批次管理复杂**: 
   - 坯布批次→染缸批次→成品布批次
   - 需要按批次核算成本（坯布成本 + 染费 + 加工费）
   
2. **色号管理特殊**:
   - 潘通色号、客户定制色号
   - 特殊色号加价核算
   - 染色配方成本管理

3. **双计量单位**:
   - 主单位：米（长度）
   - 辅单位：公斤（重量）
   - 换算关系：米数 × 克重 = 重量

4. **等级管理**:
   - 一等品、二等品、等外品
   - 不同等级不同成本
   - 等级降级损失核算

5. **委托加工**:
   - 委托印染厂加工
   - 染费结算（按米数/公斤数）
   - 加工损耗核算

### 1.2 模块核心价值

针对面料行业特点，提供：
- ✅ 批次成本核算（坯布→成品布全流程）
- ✅ 色号成本归集（染色成本单独核算）
- ✅ 双计量单位核算（米数 + 公斤数）
- ✅ 等级成本差异核算
- ✅ 委托加工染费核算
- ✅ 缸号追踪（染缸批次管理）

```
辅助核算:
- 批次号：B20260315001
- 色号：C001
- 销售数量：5,000 米
- 单位成本：18.5 元/米
- 结转成本：92,500 元

成本计算:
- 采用加权平均法
- 期初库存：3,000 米 × 18.0 元/米 = 54,000 元
- 本期入库：7,000 米 × 18.5 元/米 = 129,500 元
- 加权平均单价：(54,000 + 129,500) / (3,000 + 7,000) = 18.35 元/米
- 结转成本：5,000 米 × 18.35 元/米 = 91,750 元
```

```rust
    // 本期入库
    pub debit_quantity: Decimal,        // 借方数量（入库米数）
    pub debit_amount: Decimal,          // 借方金额（入库金额）
    
    // 本期出库
    pub credit_quantity: Decimal,       // 贷方数量（出库米数）
    pub credit_amount: Decimal,         // 贷方金额（出库金额）
    
    // 期末
    pub ending_quantity: Decimal,       // 期末数量
    pub ending_amount: Decimal,         // 期末金额
    
    // 单价计算
    pub weighted_average_price: Decimal, // 加权平均单价
}

impl FabricLedgerLine {
    // 计算加权平均单价
    pub fn calculate_weighted_average_price(&mut self) {
        let total_quantity = self.beginning_quantity + self.debit_quantity;
        let total_amount = self.beginning_amount + self.debit_amount;
        
        if total_quantity.is_zero() {
            self.weighted_average_price = Decimal::ZERO;
        } else {
            self.weighted_average_price = total_amount / total_quantity;
        }
    }
    
    // 计算出库成本（加权平均法）
    pub fn calculate_outbound_cost(&self, outbound_quantity: Decimal) -> Decimal {
        outbound_quantity * self.weighted_average_price
    }
}
```

```rust
pub fn allocate_dyeing_fee_to_batches(
    total_dyeing_fee: Decimal,
    batch_quantities: Vec<BatchQuantity>
) -> Vec<BatchDyeingFeeAllocation> {
    let total_quantity: Decimal = batch_quantities
        .iter()
        .map(|bq| bq.quantity)
        .sum();
    
    let allocation_rate = total_dyeing_fee / total_quantity;
    
    batch_quantities

            batch_no: bq.batch_no,
            color_no: bq.color_no,
            quantity: bq.quantity,
            allocation_rate,
            allocated_fee: bq.quantity * allocation_rate,
        })
        .collect()
}
```
