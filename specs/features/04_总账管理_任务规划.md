### 任务 1.7：路由配置

**任务描述**: 在 routes/mod.rs 中添加总账管理路由

**输入**: Handler 层

**输出**: `backend/src/routes/mod.rs` (更新)

**具体内容**:
```rust
// 总账管理路由
mod gl_routes {
    use super::*;
    use crate::handlers::{account_subject_handler, voucher_handler};

    pub fn routes() -> Router<Arc<AppState>> {
        Router::new()
            .nest("/subjects", subject_routes())
            .nest("/vouchers", voucher_routes())
    }

    fn subject_routes() -> Router<Arc<AppState>> {
        Router::new()
            .route("/", get(account_subject_handler::list_subjects))
            .route("/", post(account_subject_handler::create_subject))
            .route("/:id", get(account_subject_handler::get_subject))
            .route("/:id", put(account_subject_handler::update_subject))
            .route("/:id", delete(account_subject_handler::delete_subject))
    }

    fn voucher_routes() -> Router<Arc<AppState>> {
        Router::new()
            .route("/", get(voucher_handler::list_vouchers))
            .route("/", post(voucher_handler::create_voucher))
            .route("/:id", get(voucher_handler::get_voucher))
            .route("/:id", put(voucher_handler::update_voucher))
            .route("/:id", delete(voucher_handler::delete_voucher))
            .route("/:id/submit", post(voucher_handler::submit_voucher))
            .route("/:id/review", post(voucher_handler::review_voucher))
            .route("/:id/post", post(voucher_handler::post_voucher))
    }
}
```

**预计时间**: 1 小时

**验收标准**:
- [ ] 路由配置正确
- [ ] 路由前缀 `/api/v1/erp/finance/gl/`
