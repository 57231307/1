## 一、模块概述

### 1.1 模块定位
采购管理模块是 ERP 系统供应链管理的核心环节，连接供应商管理和库存管理。该模块负责管理从采购申请到采购入库的完整流程，与供应商管理、库存管理、应付管理紧密集成，形成完整的供应链闭环。

### 1.2 业务价值
- **规范采购流程**：建立标准化的采购业务流程
- **降低采购成本**：通过供应商比价、采购价格控制降低成本
- **提高采购效率**：自动化采购流程，减少人工操作
- **保证物料供应**：及时采购，保证生产物料供应
- **质量追溯**：质量问题可追溯到具体采购订单和供应商

### 1.3 核心功能
1. 采购申请管理
2. 采购订单管理
3. 采购价格管理
4. 采购入库管理
5. 采购退货管理
6. 采购质检管理
7. 采购统计分析

## 二、用户角色与使用场景

### 2.1 用户角色

#### 采购专员
- **职责**：创建采购订单、跟踪采购执行
- **核心需求**：
  - 根据库存预警创建采购订单
  - 选择合适供应商
  - 跟踪采购订单执行状态
  - 处理采购退货

#### 采购经理
- **职责**：审核采购订单、审批采购价格
- **核心需求**：
  - 审核采购订单
  - 审批采购价格
  - 查看采购统计分析
  - 供应商绩效评估

#### 仓库管理员
- **职责**：采购入库、质检
- **核心需求**：
  - 采购到货通知
  - 采购入库操作
  - 来料质量检验
  - 采购退货出库

#### 质量管理员
- **职责**：来料质量检验
- **核心需求**：
  - 来料检验记录
  - 质量判定（合格/不合格）
  - 质量异常处理
  - 质量统计分析

#### 财务人员
- **职责**：采购发票、付款
- **核心需求**：
  - 采购发票核对
  - 采购应付账款
  - 采购付款申请

| 外观 | 无瑕疵 | 少量瑕疵 | 合格 |

**业务规则**：
1. 检验单号自动生成规则：`IQC` + `年月日` + `3 位序号`
2. 必须关联入库单
3. 检验结果影响入库状态
4. 不合格品自动触发退货流程
5. 检验结果自动记录到供应商评估
6. 支持质检报告附件

**操作**：
- 创建检验记录
- 填写检验结果
- 上传检验报告
- 判定合格/不合格
- 生成质量统计

## 五、业务流程

### 5.1 标准采购流程
```
采购申请 → 申请审核 → 采购订单 → 订单审核 → 
供应商发货 → 到货质检 → 采购入库 → 入库确认 → 
更新库存 → 更新订单状态 → 采购完成
```

### 5.2 采购退货流程
```
发现质量问题 → 创建退货单 → 退货审核 → 
退货出库 → 扣减库存 → 更新订单 → 
生成应付调整 → 供应商评估扣分
```

### 5.3 采购价格管理流程
```
供应商报价 → 价格审核 → 录入系统 → 
生效执行 → 价格监控 → 价格调整
```

## 七、与其他模块的集成关系

### 7.1 与供应商管理的集成
- 采购订单必须选择已审核的供应商
- 自动记录供应商交货及时率
- 自动记录供应商来料合格率
- 质量问题自动扣减供应商评估分数

### 7.2 与库存管理的集成
- 采购入库自动增加库存
- 自动记录批次、色号、缸号等五维信息
- 双计量单位自动换算
- 采购退货自动扣减库存

### 7.3 与应付管理的集成
- 采购入库自动生成应付账款
- 采购退货自动调整应付账款
- 采购发票自动关联采购订单
- 付款自动核销应付账款

### 7.4 与质量管理的集成
- 来料检验记录共享
- 质量事故记录共享
- 质量统计分析共享
