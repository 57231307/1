## 一、面料行业核心特征

### 1.1 行业特殊性

1. **批次管理复杂**: 坯布批次 → 染缸批次 → 成品布批次
2. **色号管理特殊**: 潘通色号、客户定制色号、特殊色号加价
3. **双计量单位**: 米（长度）+ 公斤（重量），换算：米数 × 克重 = 重量
4. **等级管理**: 一等品、二等品、等外品，等级降级损失
5. **缸号管理**: 染缸编号、缸差管理、质量追溯
6. **委托加工**: 染费结算、加工损耗、超耗扣款

## 三、详细适配方案

### 3.1 产品管理模块（深度适配）

#### 新增字段
```rust
pub struct Product {
    // 面料行业字段
    pub product_type: String,             // 产品类型：坯布/成品布/辅料
    pub fabric_composition: String,       // 面料成分：65% 棉 35% 涤
    pub yarn_count: Option<String>,       // 纱支：40S
    pub density: Option<String>,          // 密度：133x72
    pub width: Option<Decimal>,           // 幅宽（cm）
    pub gram_weight: Option<Decimal>,     // 克重（g/m²）
    pub structure: Option<String>,        // 组织结构：平纹/斜纹/缎纹
    pub finish: Option<String>,           // 后整理：防水/防油/阻燃
    pub min_order_quantity: Option<Decimal>, // 最小起订量（米）
    pub lead_time: Option<i32>,           // 交货期（天）
}
```

#### 新增表：产品色号表
```sql
CREATE TABLE product_colors (
    id SERIAL PRIMARY KEY,
    product_id INTEGER NOT NULL REFERENCES products(id),
    color_no VARCHAR(50) NOT NULL,
    color_name VARCHAR(100) NOT NULL,
    pantone_code VARCHAR(50),
    color_type VARCHAR(20) DEFAULT '常规色',
    dye_formula TEXT,                     -- 染色配方（保密）
    extra_cost DECIMAL(10,2) DEFAULT 0,   -- 特殊色号加价
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

COMMENT ON TABLE product_colors IS '产品色号表';
CREATE INDEX idx_product_colors_product_id ON product_colors(product_id);
```

```sql
CREATE TABLE inventory_transactions (
    id SERIAL PRIMARY KEY,
    transaction_type VARCHAR(20) NOT NULL,
    product_id INTEGER NOT NULL REFERENCES products(id),
    warehouse_id INTEGER NOT NULL REFERENCES warehouses(id),
    batch_no VARCHAR(50) NOT NULL,
    color_no VARCHAR(50) NOT NULL,
    dye_lot_no VARCHAR(50),
    grade VARCHAR(20) DEFAULT '一等品',
    
    -- 双计量单位
    quantity_meters DECIMAL(12,2) NOT NULL,
    quantity_kg DECIMAL(12,2) NOT NULL,
    
    -- 关联单据
    source_bill_type VARCHAR(50),
    source_bill_no VARCHAR(50),
    source_bill_id INTEGER,
    
    -- 库存变化
    quantity_before_meters DECIMAL(12,2),
    quantity_before_kg DECIMAL(12,2),
    quantity_after_meters DECIMAL(12,2),
    quantity_after_kg DECIMAL(12,2),
    
    created_at TIMESTAMPTZ DEFAULT NOW()
);

COMMENT ON TABLE inventory_transactions IS '库存流水表（面料行业）';
CREATE INDEX idx_inventory_transactions_batch ON inventory_transactions(batch_no, color_no);
```
