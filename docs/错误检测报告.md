### 2.4 Decimal 类型转换错误 ❌ 严重

**影响文件**:

- 其他服务文件（约 20+ 处）

**错误描述**:
```
error[E0277]: the trait bound `rust_decimal::Decimal: From<f64>` is not satisfied
  --> src/services/supplier_service.rs:45:25
   |
45 |     let amount = Decimal::from(100.50);
   |                  ^^^^^^^^^^^^^ the trait `From<f64>` is not implemented for `rust_decimal::Decimal`
```

**错误原因**: `rust_decimal::Decimal` 未实现 `From<f64>` trait，因为浮点数到小数的转换可能丢失精度

**建议修复方案**:
```rust
// 错误写法
let value = Decimal::from(100.50);

// 正确写法 1: 使用字符串
let value = Decimal::from_str("100.50").unwrap();

// 正确写法 2: 使用 from_f64_retain（可能失败）
let value = Decimal::from_f64_retain(100.50).unwrap_or(Decimal::ZERO);

// 正确写法 3: 使用 try_from
let value = Decimal::try_from(100.50f64).unwrap_or(Decimal::ZERO);
```

**严重程度**: 🔴 严重

## 🟠 六、安全问题（10 个）

### 6.1 错误处理不完整 🟠 中等

**观察**: 多处使用 `.unwrap()` 而非适当错误处理

**错误模式**:
```rust
// 可能 panic 的代码
let value = some_option.unwrap();
let result = some_result.unwrap();
```

**建议修复**:
```rust
// 使用 match
match some_option {
    Some(value) => value,
    None => return Err(AppError::ResourceNotFound("...".to_string())),
}

// 使用 ? 操作符
let value = some_option.ok_or(AppError::BadRequest("...".to_string()))?;
```

**严重程度**: 🟠 中等（可能导致运行时 panic）

### 6.2 敏感信息日志 🟠 中等

**风险示例**:
```rust
// 可能泄露密码

// 可能泄露 token

// 只记录非敏感信息
tracing::info!("用户登录尝试：username={}", username);

```

**严重程度**: 🟠 中等（安全风险）
