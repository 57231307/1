## 四、编译问题

### 问题描述
编译失败，缺少 `protoc`（Protocol Buffer 编译器）

**错误信息**:
```
Error: Custom { kind: NotFound, error: "Could not find `protoc`" }
```

### 解决方案

#### 方案 1：安装 protoc（推荐）
**Windows 系统**:
```powershell
# 使用 winget 安装
winget install Google.ProtobufCompiler

# 或者下载二进制文件
# https://github.com/protocolbuffers/protobuf/releases
# 下载 protoc-xx.x-win64.zip
# 解压后添加到 PATH
```

#### 方案 2：临时禁用 gRPC
如果暂时不需要 gRPC 功能，可以注释掉 `build.rs` 中的 protobuf 编译代码。

#### 方案 3：使用 Docker
在 Docker 容器中编译，容器已预装 protoc。
