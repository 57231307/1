## 二、面料行业特色功能

### 2.1 产品管理

✅ **色号管理体系**
- 潘通色号支持
- 常规色/定制色分类
- 特殊色号加价（如：荧光色 +2.5 元/米）
- 染色配方管理（保密字段）

**应用场景**:
```
产品：涤棉斜纹布
├─ 色号 C001：藏青色（潘通 19-4052 TCX，常规色，不加价）
├─ 色号 C002：大红色（潘通 18-1664 TCX，常规色，不加价）
└─ 色号 C004：荧光绿（潘通 802 C，定制色，加价 2.5 元/米）
```

### 2.2 库存管理

✅ **批次 + 色号 + 缸号管理**
- 批次追溯（从成品追溯到坯布）
- 色号管理（潘通色号、客户定制色）
- 缸号追溯（染缸批次，质量追溯）
- 等级管理（一等品/二等品/等外品）

✅ **双计量单位**
- 主单位：米（长度）
- 辅单位：公斤（重量）
- 自动换算：米数 × 克重 = 重量

**应用场景**:
```
库存：涤棉斜纹布
批次：B20260315001
色号：C001（藏青色）
缸号：D20260315001
等级：一等品
数量：10,000 米 / 1,500 公斤
克重：150 g/m²
幅宽：150 cm
库位：A-01-02
```

✅ **库存流水追溯**
- 记录每一笔库存变动
- 支持批次、色号追溯
- 双计量单位同步记录

### 2.3 销售管理

✅ **色号销售**
- 订单必须录入色号
- 支持潘通色号
- 特殊色号自动加价

✅ **等级要求**
- 客户可指定等级要求
- 一等品/二等品/等外品

✅ **双计量单位**
- 订单数量：米数 + 公斤数
- 已发货数量：米数 + 公斤数

**应用场景**:
```
销售订单：SO20260315001
产品：涤棉斜纹布
色号：C001（藏青色）
等级要求：一等品
数量：5,000 米 / 750 公斤
基价：18.00 元/米
色号加价：0.00 元
最终单价：18.00 元/米
```

### 2.4 总账管理

✅ **辅助核算（面料行业）**
- 批次辅助核算
- 色号辅助核算
- 缸号辅助核算
- 等级辅助核算
- 车间辅助核算

✅ **双计量单位核算**
- 科目支持米数 + 公斤数
- 凭证自动记录双计量单位
- 账簿支持数量金额式

**应用场景**:
```
凭证：记字 [2026]03 第 0001 号
业务：成品布完工入库

借：1405.02 库存商品 - 成品布
    数量：10,000 米 / 1,500 公斤
    金额：180,000 元
    批次：B20260315001
    色号：C001
    缸号：D20260315001
    
  贷：5001 生产成本
      金额：180,000 元
```

   - 潘通色号标准化
