## 🔍 详细检测结果

### 1. 后端编译错误分析

#### 错误类型 A：RustDecimal 类型未定义（8 个文件）

**错误代码**: `E0412 - cannot find type 'RustDecimal' in this scope`

**影响范围**:
- `purchase_receipt.rs` - 3 处
- `ap_invoice.rs` - 6 处
- `ap_payment.rs` - 已正确导入别名
- `ap_payment_request.rs` - 已正确导入别名
- `ap_payment_request_item.rs` - 已正确导入别名
- `ap_verification.rs` - 2 处
- `ap_verification_item.rs` - 2 处
- `ap_reconciliation.rs` - 6 处

**根本原因**: 
部分模型文件直接使用 `RustDecimal` 类型但未导入，而其他文件使用 `use rust_decimal::Decimal as RustDecimal;` 导入别名。

**修复方案**:
```rust
// 在文件顶部添加
use rust_decimal::Decimal;

// 替换所有 RustDecimal 为 Decimal
```

#### 错误类型 B：Set 函数未导入（Service 层）

**错误代码**: `E0425 - cannot find function, tuple struct or tuple variant 'Set' in this scope`

**影响范围**:
- `purchase_receipt_service.rs` - 100+ 处错误

**根本原因**:
SeaORM 的 ActiveModel 需要使用 `ActiveValue::Set` 来设置字段值，但缺少导入。

**修复方案**:
```rust
use sea_orm::{
    ActiveModelTrait, ColumnTrait, DatabaseConnection, EntityTrait, QueryFilter,
    QueryOrder, TransactionTrait, ActiveValue::Set,  // 添加这一行
};
```

#### 错误类型 C：NaiveDate 类型未导入

**影响范围**:
- `ap_verification.rs`
- `ap_reconciliation.rs`

**修复方案**:
```rust
use chrono::NaiveDate;
```

### 2. 前端编译错误分析

#### 错误类型 A：Yew Router API 变更（20 个页面文件）

**错误代码**: 
- `E0277 - the trait bound 'yew_router::BrowserRouter: yew::Component' is not satisfied`
- `E0599 - no method named 'navigator' found`

**影响范围**: 所有 20 个页面组件文件

**根本原因**:
Yew 从 0.20 升级到 0.21 后，Router API 发生重大变更：
- `use_navigator()` hook 行为变更
- `BrowserRouter` 不再直接实现 Component trait
- 需要使用可选绑定代替 `unwrap()`

**修复方案**:
```rust
// 旧代码
let navigator = use_navigator().unwrap();
navigator.push(Route::ProductList);

// 新代码
let navigator = use_navigator();
if let Some(nav) = navigator {
    nav.push(Route::ProductList);
}
```

#### 错误类型 B：缺少 JsCast trait 导入

**错误代码**: `E0599 - no method named 'dyn_into' found`

**修复方案**:
```rust
use wasm_bindgen::cast::JsCast;
```

#### 错误类型 C：Service 返回类型不匹配

**影响文件**: `warehouse_service.rs`

**问题**: 期望 `Result<Warehouse, String>` 但返回 `Result<Value, String>`

**修复方案**: 调整泛型参数，确保类型匹配
