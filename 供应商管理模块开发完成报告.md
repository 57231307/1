## 五、核心功能特性

### 5.1 供应商编码自动生成

```rust
/// 生成供应商编码
/// 格式：SUP + 年月日 + 3 位序号
/// 示例：SUP20260315001
pub async fn generate_supplier_code(&self) -> Result<String, AppError> {
    let today = Utc::now().format("%Y%m%d").to_string();
    let prefix = format!("SUP{}", today);
    
    // 查询今日已有数量
    let count = supplier::Entity::find()
        .filter(supplier::Column::SupplierCode.starts_with(&prefix))
        .count(&*self.db)
        .await?;
    
    Ok(format!("{}{:03}", prefix, count + 1))
}
```

### 5.2 供应商分级管理

- **A 级（战略供应商）**: 90-100 分，优先采购、免检、月结
- **B 级（合格供应商）**: 75-89 分，正常采购、抽检、月结
- **C 级（考察供应商）**: 60-74 分，限制采购、全检、现结
- **D 级（不合格供应商）**: 60 分以下，暂停采购、列入黑名单

### 5.3 供应商评估体系

**5 个维度，总分 100 分**：
1. 质量水平（35 分）- 来料合格率、质量事故、质量改进
2. 交货能力（25 分）- 交货及时率、订单完成率
3. 价格水平（20 分）- 价格竞争力、价格稳定性
4. 服务水平（15 分）- 响应速度、售后服务、配合度
5. 技术能力（5 分）- 研发能力、技术支持

### 6.2 分页查询

支持复杂的筛选、排序和分页：
```rust
pub async fn list_suppliers(
    &self,
    params: SupplierQueryParams,
) -> Result<PaginatedResponse<supplier::Model>, AppError> {
    // 筛选
    if let Some(supplier_type) = params.supplier_type {
        query = query.filter(supplier::Column::SupplierType.eq(supplier_type));
    }
    
    // 关键词搜索
    if let Some(keyword) = params.keyword {
        query = query.filter(
            supplier::Column::SupplierName.contains(&keyword)
                .or(supplier::Column::SupplierCode.contains(&keyword))
        );
    }
    
    // 排序
    query = match sort_by.as_str() {
        "supplier_code" => query.order_by_desc(supplier::Column::SupplierCode),
        "supplier_name" => query.order_by_desc(supplier::Column::SupplierName),
        _ => query.order_by_desc(supplier::Column::CreatedAt),
    };
    
    // 分页
    let paginator = query.paginate(&*self.db, page_size);
    let items = paginator.fetch_page(page - 1).await?;
    
    Ok(PaginatedResponse { items, page, page_size, total })
}
```
