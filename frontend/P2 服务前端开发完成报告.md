## 🚀 下一步操作

在 IDE 中打开项目，运行 `cargo check`,根据错误提示添加 `.clone()`:

```bash

```

### 2. 安装 Trunk (如果未安装)
```bash

```

### 3. 运行开发服务器
```bash
trunk serve
```

访问 http://localhost:8080 查看效果

### 4. 生产打包
```bash
trunk build --release
```
