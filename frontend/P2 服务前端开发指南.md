# P2 级服务前端开发指南

## 📋 概述

本文档描述如何使用 Yew + Trunk 技术栈开发面料 ERP 系统 P2 级服务的前端界面。

**技术栈**:
- **框架**: Yew 0.21 (Rust WebAssembly 框架)
- **路由**: yew-router 0.17
- **HTTP 客户端**: gloo-net 0.4
- **样式**: Tailwind CSS
- **构建工具**: Trunk

**前端基础路径**: `/frontend`  
**API 基础路径**: `/api/v1/erp`

## 📊 API 调用示例

### GET 请求（带参数）

```rust
let params = vec![
    ("page", "0".to_string()),
    ("page_size", "10".to_string()),
];

let query = if params.is_empty() {
    String::new()
} else {
    format!("?{}", params.iter()
        .map(|(k, v)| format!("{}={}", k, v))
        .collect::<Vec<_>>()
        .join("&"))
};

let response: serde_json::Value = ApiService::get(&format!("/sales/prices{}", query)).await?;
```

### POST 请求

```rust
let body = serde_json::json!({
    "product_id": 1,
    "supplier_id": 2,
    "price": "100.00",
    "effective_date": "2026-03-16",
});

let response: serde_json::Value = ApiService::post("/sales/prices", &body).await?;
```

### PUT 请求

```rust
let body = serde_json::json!({
    "price": "105.00",
    "status": "approved",
});

let response: serde_json::Value = ApiService::put("/sales/prices/1", &body).await?;
```

### DELETE 请求

```rust
ApiService::delete("/sales/prices/1").await?;
```
