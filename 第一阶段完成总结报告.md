```sql
CREATE TABLE product_colors (
    id SERIAL PRIMARY KEY,
    product_id INTEGER NOT NULL,
    color_no VARCHAR(50) NOT NULL,
    color_name VARCHAR(100) NOT NULL,
    pantone_code VARCHAR(50),
    color_type VARCHAR(20) DEFAULT '常规色',
    dye_formula TEXT,              -- 染色配方（保密）
    extra_cost DECIMAL(10,2),      -- 特殊色号加价
    is_active BOOLEAN DEFAULT true
);
```

**会计科目表** (`account_subjects`):
```sql
CREATE TABLE account_subjects (
    id SERIAL PRIMARY KEY,
    code VARCHAR(50) NOT NULL,
    name VARCHAR(200) NOT NULL,
    level INTEGER NOT NULL,
    assist_batch BOOLEAN,          -- 批次辅助核算
    assist_color_no BOOLEAN,       -- 色号辅助核算
    enable_dual_unit BOOLEAN,      -- 双计量单位
    primary_unit VARCHAR(20),      -- 米
    secondary_unit VARCHAR(20)     -- 公斤
);
```

## 三、面料行业特色功能

### 3.1 产品管理

✅ **色号管理体系**:
- 潘通色号标准化
- 常规色/定制色分类
- 特殊色号加价（如：荧光色 +2.5 元/米）
- 染色配方管理（保密字段）

**应用场景**:
```
产品：涤棉斜纹布
├─ C001 藏青色（潘通 19-4052 TCX，常规色，不加价）
├─ C002 大红色（潘通 18-1664 TCX，常规色，不加价）
└─ C004 荧光绿（潘通 802 C，定制色，加价 2.5 元/米）
```

---

### 3.2 库存管理

✅ **批次 + 色号 + 缸号管理**:
- 批次追溯（坯布→染缸→成品布）
- 色号管理（潘通色号、定制色）
- 缸号追溯（染缸批次，质量追溯）
- 等级管理（一等品/二等品/等外品）

✅ **双计量单位**:
- 主单位：米（长度）
- 辅单位：公斤（重量）
- 自动换算：米数 × 克重 = 重量

✅ **库存流水追溯**:
- 记录每一笔库存变动
- 支持批次、色号追溯
- 双计量单位同步记录

**应用场景**:
```
库存：涤棉斜纹布
批次：B20260315001
色号：C001（藏青色）
缸号：D20260315001
等级：一等品
数量：10,000 米 / 1,500 公斤
克重：150 g/m²
库位：A-01-02
```
